# Glass Effect System

The Glass Effect System is a comprehensive liquid glass implementation for modern React applications, built with Next.js, Tailwind CSS, and GSAP animations.

## Overview

Our Glass Effect System provides a complete liquid glass morphism solution with:

- **React Components**: Modern component-based architecture
- **GSAP Animations**: Smooth, performant animations
- **Tailwind Integration**: Utility-first CSS framework
- **Accessibility**: Built-in accessibility features
- **Performance**: Optimized for production

## Core Components

### GlassEffect

The base component for all glass effects:

```tsx
import { GlassEffect } from '@/components/ui';

<GlassEffect variant='card' intensity='medium' glow>
  <p>Content with liquid glass effect</p>
</GlassEffect>;
```

**Props:**

- `variant`: `'default' | 'card' | 'modal' | 'button' | 'overlay' | 'widget'`
- `intensity`: `'light' | 'medium' | 'heavy' | 'ultra'`
- `borderRadius`: `'sm' | 'md' | 'lg' | 'xl' | 'full'`
- `shadow`: `'none' | 'sm' | 'md' | 'lg' | 'xl'`
- `hover`: `boolean` - Enable hover effects
- `animated`: `boolean` - Enable GSAP animations
- `glow`: `boolean` - Enable glow effect

### GlassCard

Pre-configured card component with liquid glass effect:

```tsx
import { GlassCard } from '@/components/ui';

<GlassCard
  header={<h3>Card Title</h3>}
  footer={<button>Action</button>}
  padding='lg'
  spacing='md'
>
  <p>Card content with automatic glass styling</p>
</GlassCard>;
```

### GlassButton

Interactive button with liquid glass effect:

```tsx
import { GlassButton } from '@/components/ui';

<GlassButton
  variant='primary'
  size='lg'
  loading={false}
  onClick={() => alert('Clicked!')}
>
  Click Me
</GlassButton>;
```

### GlassBackground

Full-page background with animated glass orbs:

```tsx
import { GlassBackground } from '@/components/ui';

<GlassBackground variant='gradient' animated>
  <div>Your page content</div>
</GlassBackground>;
```

## Animation System

All components include GSAP-powered animations:

- **Hover Effects**: 3D transforms and magnetic attraction
- **Focus States**: Accessibility-compliant focus rings
- **Entrance Animations**: Smooth reveal on scroll
- **Interactive Feedback**: Ripple effects and state changes

## Accessibility

The system includes comprehensive accessibility features:

- **Reduced Motion**: Respects `prefers-reduced-motion`
- **High Contrast**: Adapts to `prefers-contrast`
- **Transparency**: Supports `prefers-reduced-transparency`
- **Keyboard Navigation**: Full keyboard support
- **Screen Readers**: Proper ARIA attributes

## Performance

Optimized for production:

- **Hardware Acceleration**: GPU-optimized transforms
- **Lazy Loading**: Components load only when needed
- **Tree Shaking**: Only used components are bundled
- **Memory Management**: Automatic cleanup of animations

## Migration from Legacy

If you're migrating from legacy CSS classes:

| Old Class              | New Component   | Usage                             |
| ---------------------- | --------------- | --------------------------------- |
| `.liquid-glass`        | `<GlassEffect>` | `<GlassEffect variant="default">` |
| `.liquid-glass-card`   | `<GlassCard>`   | `<GlassCard>`                     |
| `.liquid-glass-button` | `<GlassButton>` | `<GlassButton>`                   |
| `.liquid-glass-thin`   | `<GlassEffect>` | `<GlassEffect intensity="light">` |
| `.liquid-glass-thick`  | `<GlassEffect>` | `<GlassEffect intensity="heavy">` |

## Best Practices

1. **Use appropriate variants** for different content types
2. **Limit nesting** to avoid performance issues
3. **Test accessibility** with keyboard navigation
4. **Consider performance** on lower-end devices
5. **Follow Apple HIG** guidelines for consistency
