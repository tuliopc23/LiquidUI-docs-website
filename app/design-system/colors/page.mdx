## Semantic Colors

Specifically tailored for components to convey meaning and intention.

### Usage
```tsx
import { colors } from 'liquidify/tokens'

const primary = colors.blue // #007AFF for primary actions
const warning = colors.orange // #FF9500 for warnings
```

## Custom Themes

All colors can be customized to create unique themes while maintaining accessibility and brand guidelines.

### Example
```tsx
const customTheme = {
  ...colors,
  primary: '#4A90E2', // Custom brand primary color
}
```
