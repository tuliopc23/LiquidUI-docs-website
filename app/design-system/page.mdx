# Design System

The Liquidify Design System is built on Apple's Human Interface Guidelines with cutting-edge Liquid Glass material principles introduced at WWDC 2025.

## Overview

Our design system provides a comprehensive foundation for building beautiful, accessible, and performant user interfaces that feel native across all Apple platforms. Every token, component, and pattern is carefully crafted to deliver exceptional user experiences.

## Core Principles

### 1. Liquid Glass Material
The revolutionary Liquid Glass material adapts to content and context, providing:
- **Content-aware transparency** that adjusts to background
- **Dynamic blur effects** that respond to motion
- **Intelligent contrast** for optimal readability
- **Fluid morphing** between states and contexts

### 2. Apple Human Interface Guidelines
Every aspect follows Apple's design principles:
- **Clarity** - Clear visual communication and hierarchy
- **Deference** - Content takes precedence over chrome
- **Depth** - Realistic layering and meaningful motion
- **Accessibility** - Inclusive design for all users

### 3. Performance First
Optimized for real-world usage:
- **Hardware acceleration** for smooth 120fps animations
- **Bundle efficiency** with tree-shakable components
- **Memory optimization** for complex interactions
- **Battery conscious** animations and effects

## Implementation

### Installation
```bash
npm install liquidify
```

### Theme Setup
```tsx
import { ThemeProvider } from 'liquidify'
import 'liquidify/styles'

export default function App({ children }) {
  return (
    <ThemeProvider>
      {children}
    </ThemeProvider>
  )
}
```

### Component Usage
```tsx
import { GlassButton, GlassCard } from 'liquidify'

export default function Example() {
  return (
    <GlassCard className="p-6">
      <h2 className="hig-title-2 mb-4">Welcome to Liquidify</h2>
      <p className="hig-body mb-6">
        Experience the future of interface design.
      </p>
      <GlassButton variant="primary">
        Get Started
      </GlassButton>
    </GlassCard>
  )
}
```
