## Code Examples

### Basic Particle Emitter

```typescript
import { ParticleEmitter, Vector2D, PARTICLE_PRESETS } from 'liquidify';

const sparkEmitter = new ParticleEmitter({
  ...PARTICLE_PRESETS.sparks,
  position: new Vector2D(200, 200),
});

function animate() {
  sparkEmitter.update(16);
  sparkEmitter.render(ctx);
  requestAnimationFrame(animate);
}
```

### Interactive Mouse Particles

```typescript
canvas.addEventListener('mousemove', e => {
  const rect = canvas.getBoundingClientRect();
  const mousePos = new Vector2D(e.clientX - rect.left, e.clientY - rect.top);

  emitter.setPosition(mousePos);
  emitter.burst(5); // Emit 5 particles at mouse position
});
```

### Custom Particle Effect

```typescript
const customEmitter = new ParticleEmitter({
  position: new Vector2D(300, 300),
  rate: 80,
  particleLifetime: 1500,
  velocity: new Vector2D(0, -30),
  velocityVariation: new Vector2D(60, 20),
  size: 3,
  sizeVariation: 1,
  color: '#ff6b6b',
  gravity: new Vector2D(0, 20),
  fadeOut: true,
  shrink: true,
});
```

## Performance Tips

1. **Limit particle count**: Keep under 1000 active particles
2. **Use culling**: Remove off-screen particles
3. **Batch rendering**: Group similar particles
4. **Object pooling**: Reuse particle objects

## Customization Options

### Particle Properties

- **Size**: Control particle dimensions
- **Color**: Set particle colors and opacity
- **Lifetime**: Adjust how long particles live
- **Velocity**: Control initial speed and direction

### Physics Properties

- **Gravity**: Apply downward force
- **Air Resistance**: Simulate drag
- **Acceleration**: Constant force application
- **Collision**: Boundary interactions

### Visual Effects

- **Fade Out**: Gradual opacity reduction
- **Shrink**: Size reduction over time
- **Color Transitions**: Animate color changes
- **Blend Modes**: Canvas composition modes
