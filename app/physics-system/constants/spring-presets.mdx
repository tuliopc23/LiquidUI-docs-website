# SPRING_PRESETS

The `SPRING_PRESETS` object provides pre-configured spring settings for common animation patterns and feels in the Liquidify physics system.

## Overview

`SPRING_PRESETS` contains carefully tuned spring configurations that achieve specific animation characteristics. These presets save time and ensure consistent, professional-feeling animations across your application.

## Import

```typescript
import { SPRING_PRESETS } from 'liquidify';
```

## Available Presets

### `gentle`

Smooth, gentle motion with minimal overshoot. Perfect for subtle UI animations.

```typescript
SPRING_PRESETS.gentle = {
  stiffness: 100,
  damping: 10,
  mass: 1,
  precision: 0.001,
};
```

**Characteristics:**

- Smooth acceleration and deceleration
- Minimal bounce or overshoot
- Settles quickly without oscillation
- Feels natural and refined

**Use Cases:**

- Card hover effects
- Menu transitions
- Subtle element movements
- Professional UI animations

## Usage Examples

### Basic Usage

```typescript
import { SpringPhysics, SPRING_PRESETS } from 'liquidify';

// Create springs with different presets
const gentleSpring = new SpringPhysics(SPRING_PRESETS.gentle);
const snappySpring = new SpringPhysics(SPRING_PRESETS.snappy);
const bouncySpring = new SpringPhysics(SPRING_PRESETS.bouncy);
```

### React Hook Usage

```typescript
import { useSpringPhysics, SPRING_PRESETS } from 'liquidify';

function AnimatedButton() {
  const { position, setTarget } = useSpringPhysics(SPRING_PRESETS.snappy);

  return (
    <button
      style={{
        transform: `scale(${1 + position * 0.1})`,
      }}
      onMouseDown={() => setTarget(1)}
      onMouseUp={() => setTarget(0)}
    >
      Press me
    </button>
  );
}
```

### Conditional Presets

```typescript
import { SPRING_PRESETS } from 'liquidify';

function getSpringPreset(context: string) {
  switch (context) {
    case 'button':
      return SPRING_PRESETS.snappy;
    case 'card':
      return SPRING_PRESETS.gentle;
    case 'game':
      return SPRING_PRESETS.bouncy;
    case 'drag':
      return SPRING_PRESETS.stiff;
    default:
      return SPRING_PRESETS.gentle;
  }
}

const spring = new SpringPhysics(getSpringPreset('button'));
```

### Custom Presets Based on Existing Ones


### Dynamic Preset Selection

```typescript
import { SPRING_PRESETS } from 'liquidify';

function DynamicSpringComponent() {
  const [springType, setSpringType] = useState('gentle');
  const spring = new SpringPhysics(SPRING_PRESETS[springType]);

  const handlePresetChange = (newPreset: string) => {
    setSpringType(newPreset);
    spring.setProperties(SPRING_PRESETS[newPreset]);
  };

  return (
    <div>
      <select onChange={(e) => handlePresetChange(e.target.value)}>
        <option value="gentle">Gentle</option>
        <option value="snappy">Snappy</option>
        <option value="bouncy">Bouncy</option>
        <option value="stiff">Stiff</option>
        <option value="wobbly">Wobbly</option>
      </select>
    </div>
  );
}
```

## Visual Characteristics

### Motion Curves

Each preset creates a distinct motion curve:

- **Gentle**: Smooth S-curve with gradual acceleration/deceleration
- **Snappy**: Quick acceleration with slight overshoot
- **Bouncy**: Multiple oscillations with decreasing amplitude
- **Stiff**: Linear motion with abrupt stop
- **Wobbly**: Sine-wave-like oscillations
- **Elastic**: Rubber-band stretch and release
- **Smooth**: Perfect ease-in-out without overshoot
- **Quick**: Sharp acceleration with quick settle
- **Heavy**: Slow start with momentum
- **Delicate**: Gentle curve with minimal energy

### Timing Characteristics

- **Fastest**: `stiff`, `quick`
- **Fast**: `snappy`, `bouncy`
- **Medium**: `gentle`, `smooth`, `elastic`
- **Slow**: `wobbly`, `heavy`, `delicate`

### Overshoot Levels

- **No overshoot**: `stiff`, `smooth`
- **Minimal overshoot**: `gentle`, `quick`, `delicate`
- **Moderate overshoot**: `snappy`, `elastic`
- **High overshoot**: `bouncy`, `wobbly`

## Best Practices

1. **Match preset to context**: Use `gentle` for subtle effects, `snappy` for interactions
2. **Maintain consistency**: Use the same preset for similar interactions
3. **Consider user expectations**: Match the animation feel to your brand
4. **Test across devices**: Some presets may feel different on various devices
5. **Customize when needed**: Extend presets rather than creating from scratch

## Accessibility Considerations

Respect user preferences for reduced motion:

```typescript
import { SPRING_PRESETS } from 'liquidify';

const getAccessiblePreset = (originalPreset: any) => {
  const prefersReducedMotion = window.matchMedia(
    '(prefers-reduced-motion: reduce)'
  ).matches;

  if (prefersReducedMotion) {
    return {
      ...originalPreset,
      stiffness: 400,
      damping: 30,
      mass: 1,
    };
  }

  return originalPreset;
};

const spring = new SpringPhysics(getAccessiblePreset(SPRING_PRESETS.bouncy));
```
