## Overview

Liquidify provides a comprehensive set of UI components designed with glassmorphism aesthetics and modern interaction patterns. Each component is built with:

- **Accessibility** in mind
- **Performance** optimizations
- **TypeScript** support
- **Customizable** theming

## Component Categories

### Layout Components

- Grid
- Container
- Stack
- Flex

### Interactive Components

- Button
- Input
- Select
- Modal

### Display Components

- Card
- Badge
- Avatar
- Tooltip

### Navigation Components

- Navbar
- Sidebar
- Breadcrumb
- Pagination
