# Glass Toggle Button

Toggle between two states using the GlassToggleButton, with smooth transitions and advanced styling.

## Overview

The `GlassToggleButton` component provides a visually intuitive way to toggle between on/off states or select/deselect options. It supports advanced Liquid Glass effects and is designed to be accessible, with ARIA attributes and smooth animations.

## Installation

```bash
npm install liquidify
```

## Basic Usage

```tsx
import { GlassToggleButton } from 'liquidify'
import { Star } from 'lucide-react'

function App() {
  const [favorited, setFavorited] = useState(false)

  return (
    <GlassToggleButton
      pressed={favorited}
      onPressedChange={setFavorited}
    >
      <Star size={16} />
      {favorited ? 'Favorited' : 'Add to Favorites'}
    </GlassToggleButton>
  )
}
```

## Advanced Example

```tsx
import { useState } from 'react'
import { GlassToggleButton } from 'liquidify'
import { Star, Sun, Moon } from 'lucide-react'

function ToggleButtonDemo() {
  const [theme, setTheme] = useState('light')
  const [favorited, setFavorited] = useState(false)

  const toggleTheme = () => {
    setTheme(theme === 'light' ? 'dark' : 'light')
  }

  return (
    <div className="space-y-6 p-6">
      <section>
        <h3 className="hig-title-3 mb-4">Theme Toggle</h3>
        <GlassToggleButton
          pressed={theme === 'dark'}
          onPressedChange={toggleTheme}
        >
          {theme === 'dark' ? <Moon size={16} /> : <Sun size={16} />}
          {theme === 'dark' ? 'Dark Mode' : 'Light Mode'}
        </GlassToggleButton>
      </section>

      <section>
        <h3 className="hig-title-3 mb-4">Favorite Toggle</h3>
        <GlassToggleButton
          pressed={favorited}
          onPressedChange={setFavorited}
        >
          <Star size={16} />
          {favorited ? 'Favorited' : 'Add to Favorites'}
        </GlassToggleButton>
      </section>
    </div>
  )
}
```

## API Reference

### Props

| Prop              | Type                      | Default | Description                    |
| ----------------- | ------------------------- | ------- | ------------------------------ |
| `pressed`         | `boolean`                 | `false` | Whether button is pressed/on   |
| `onPressedChange` | `(pressed: boolean) => void` | -    | Handler for state changes      |
| `disabled`        | `boolean`                 | `false` | Disables the toggle button     |
| `size`            | `'sm' \| 'md' \| 'lg'`    | `'md'`  | Size variant                   |
| `variant`         | `'default' \| 'outline'`  | `'default'` | Visual style variant       |
| `children`        | `ReactNode`               | -       | Button content                 |

## States

- **Default**: Unpressed/off state
- **Pressed**: Active/on state with enhanced styling
- **Disabled**: Non-interactive state
- **Loading**: Processing state with spinner

## Accessibility

- **ARIA Support**: `aria-pressed` attribute for state
- **Keyboard Navigation**: Space and Enter keys
- **Screen Reader**: Clear state announcements

## Best Practices

- Use for binary state toggles
- Provide clear visual feedback
- Include descriptive labels
- Consider context and user expectations
