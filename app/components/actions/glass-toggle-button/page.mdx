# Glass Toggle Button

Toggle between two states using the GlassToggleButton, with smooth transitions and advanced styling.

## Overview

The `GlassToggleButton` component provides a visually intuitive way to toggle between on/off states or select/deselect options. It supports advanced Liquid Glass effects and is designed to be accessible, with ARIA attributes and smooth animations powered by GSAP.

## Installation

```bash
npm install liquidify
```

## API Reference

### Props

| Prop         | Type                         | Default | Description                        |
| ------------ | ---------------------------- | ------- | ---------------------------------- |
| `pressed`    | `boolean`                    | `false` | Current toggle state               |
| `onToggle`   | `(pressed: boolean) =e void` | -       | Callback when toggle state changes |
| `disabled`   | `boolean`                    | `false` | Disables interaction               |
| `loading`    | `boolean`                    | `false` | Shows loading spinner              |
| `aria-label` | `string`                     | -       | Accessibility label (required)     |
| `className`  | `string`                     | -       | Additional CSS classes             |

## Accessibility

- **Semantic Roles**: Uses `button` role with `aria-pressed` for toggle state
- **Keyboard Accessible**: Activates with `Space` or `Enter`
- **Visual Feedback**: Clear state indication

## Best Practices

- **Use clear ARIA labels** for all toggle buttons
- **Visual consistency**: Match toggle icon size to button size
