# Glass Icon Button

Compact, touch-optimized buttons designed specifically for icon-only actions with advanced Liquid Glass effects.

## Overview

The `GlassIconButton` component provides a streamlined interface for icon-based actions while maintaining Apple's 44pt minimum touch target guidelines. Perfect for toolbars, navigation elements, and quick actions where space is at a premium.

## Installation

```bash
npm install liquidify
```

## Basic Usage

```tsx
import { GlassIconButton } from 'liquidify';
import { Heart } from 'lucide-react';

function App() {
  return (
    <GlassIconButton aria-label='Add to favorites'>
      <Heart size={20} />
    </GlassIconButton>
  );
}
```

## Interactive Demo

```tsx
import { useState } from 'react';
import { GlassIconButton } from 'liquidify';
import {
  Heart,
  Share,
  Download,
  Settings,
  MoreVertical,
  Play,
  Pause,
  Volume2,
  VolumeX,
} from 'lucide-react';

export default function IconButtonDemo() {
  const [liked, setLiked] = useState(false);
  const [playing, setPlaying] = useState(false);
  const [muted, setMuted] = useState(false);
  const [loading, setLoading] = useState(false);

  const handleDownload = async () => {
    setLoading(true);
    await new Promise(resolve => setTimeout(resolve, 2000));
    setLoading(false);
  };

  return (
    <div className='space-y-8 p-6'>
      {/* Variants */}

        <h3 className='hig-title-3 mb-4'>Variants</h3>
        <div className='flex gap-4'>

            <Heart size={20} />
          </GlassIconButton>
          <GlassIconButton variant='secondary' aria-label='Secondary action'>
            <Share size={20} />
          </GlassIconButton>
          <GlassIconButton variant='tertiary' aria-label='Tertiary action'>
            <Download size={20} />
          </GlassIconButton>
          <GlassIconButton variant='ghost' aria-label='Ghost action'>
            <Settings size={20} />
          </GlassIconButton>
          <GlassIconButton variant='destructive' aria-label='Delete'>
            <MoreVertical size={20} />
          </GlassIconButton>
        </div>
      </section>

      {/* Sizes */}
      <section>
        <h3 className='hig-title-3 mb-4'>Sizes</h3>
        <div className='flex items-center gap-4'>
          <GlassIconButton size='xs' aria-label='Extra small'>
            <Heart size={12} />
          </GlassIconButton>
          <GlassIconButton size='sm' aria-label='Small'>
            <Heart size={14} />
          </GlassIconButton>
          <GlassIconButton size='md' aria-label='Medium'>
            <Heart size={16} />
          </GlassIconButton>
          <GlassIconButton size='lg' aria-label='Large'>
            <Heart size={20} />
          </GlassIconButton>
          <GlassIconButton size='xl' aria-label='Extra large'>
            <Heart size={24} />
          </GlassIconButton>
        </div>
      </section>

      {/* Interactive States */}
      <section>
        <h3 className='hig-title-3 mb-4'>Interactive States</h3>
        <div className='flex gap-4'>
          <GlassIconButton
            variant='primary'
            aria-label={liked ? 'Remove from favorites' : 'Add to favorites'}
            onClick={() => setLiked(!liked)}
            className={liked ? 'text-red-500' : ''}
          >
            <Heart size={20} fill={liked ? 'currentColor' : 'none'} />
          </GlassIconButton>

          <GlassIconButton
            variant='secondary'
            aria-label={playing ? 'Pause' : 'Play'}
            onClick={() => setPlaying(!playing)}
          >
            {playing ? <Pause size={20} /> : <Play size={20} />}
          </GlassIconButton>

          <GlassIconButton
            variant='tertiary'
            aria-label={muted ? 'Unmute' : 'Mute'}
            onClick={() => setMuted(!muted)}
          >
            {muted ? <VolumeX size={20} /> : <Volume2 size={20} />}
          </GlassIconButton>
        </div>
      </section>

      {/* Loading State */}
      <section>
        <h3 className='hig-title-3 mb-4'>Loading State</h3>
        <GlassIconButton
          variant='primary'
          loading={loading}
          disabled={loading}
          aria-label='Download file'
          onClick={handleDownload}
        >
          <Download size={20} />
        </GlassIconButton>
      </section>

      {/* Disabled State */}
      <section>
        <h3 className='hig-title-3 mb-4'>Disabled State</h3>
        <div className='flex gap-4'>
          <GlassIconButton
            variant='primary'
            disabled
            aria-label='Disabled primary'

            <Heart size={20} />
          </GlassIconButton>
          <GlassIconButton
            variant='secondary'
            disabled
            aria-label='Disabled secondary'
          >
            <Share size={20} />

        </div>
      </section>
    </div>

}
```

## API Reference

### Props

| Prop             | Type                                                                 | Default       | Description                    |
| ---------------- | -------------------------------------------------------------------- | ------------- | ------------------------------ |
| `variant`        | `'primary' \| 'secondary' \| 'tertiary' \| 'ghost' \| 'destructive'` | `'secondary'` | Visual style variant           |
| `size`           | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'`                               | `'md'`        | Button size                    |
| `loading`        | `boolean`                                                            | `false`       | Shows loading spinner          |
| `disabled`       | `boolean`                                                            | `false`       | Disables interaction           |
| `hapticFeedback` | `'light' \| 'medium' \| 'heavy' \| 'selection' \| 'none'`            | `'light'`     | Haptic feedback intensity      |
| `magneticHover`  | `boolean`                                                            | `true`        | Enables magnetic hover effect  |
| `liquidGlass`    | `boolean`                                                            | `true`        | Enables Liquid Glass material  |
| `children`       | `ReactNode`                                                          | -             | Icon content (required)        |
| `onClick`        | `(event: MouseEvent) => void`                                        | -             | Click handler                  |
| `onHover`        | `(hovered: boolean) => void`                                         | -             | Hover state change handler     |
| `aria-label`     | `string`                                                             | -             | Accessibility label (required) |
| `className`      | `string`                                                             | -             | Additional CSS classes         |

### Size Specifications

| Size | Dimensions | Icon Size | Touch Target |
| ---- | ---------- | --------- | ------------ |
| `xs` | 24×24px    | 12px      | 44×44px      |
| `sm` | 32×32px    | 14px      | 44×44px      |
| `md` | 40×40px    | 16px      | 44×44px      |
| `lg` | 48×48px    | 20px      | 48×48px      |
| `xl` | 56×56px    | 24px      | 56×56px      |

_Note: All sizes maintain minimum 44pt touch targets for accessibility compliance._

## Variants

### Primary

High-emphasis actions that are primary to your app's flow.

```tsx
<GlassIconButton variant='primary' aria-label='Create new'>
  <Plus size={20} />
</GlassIconButton>
```

### Secondary

Medium-emphasis actions that are important but not primary.

```tsx
<GlassIconButton variant='secondary' aria-label='Edit'>
  <Edit size={20} />
</GlassIconButton>
```

### Tertiary

Low-emphasis actions or supplementary information.

```tsx
<GlassIconButton variant='tertiary' aria-label='More info'>
  <Info size={20} />
</GlassIconButton>
```

### Ghost

Minimal visual weight for subtle actions.

```tsx
<GlassIconButton variant='ghost' aria-label='Close'>
  <X size={20} />
</GlassIconButton>
```

### Destructive

Actions that delete, remove, or cause data loss.

```tsx
<GlassIconButton variant='destructive' aria-label='Delete'>
  <Trash2 size={20} />
</GlassIconButton>
```

## Icon Guidelines

### Recommended Icon Libraries

#### Lucide React (Preferred)

```tsx
import { Heart, Share, Download } from 'lucide-react';

<GlassIconButton aria-label='Like'>
  <Heart size={20} />
</GlassIconButton>;
```

#### SF Symbols (iOS/macOS)

```tsx
import { SFSymbol } from '@sf-symbols/react';

<GlassIconButton aria-label='Favorite'>
  <SFSymbol name='heart' size={20} />
</GlassIconButton>;
```

#### Heroicons

```tsx
import { HeartIcon } from '@heroicons/react/24/outline';

<GlassIconButton aria-label='Like'>
  <HeartIcon className='w-5 h-5' />
</GlassIconButton>;
```

### Icon Sizing

Match icon size to button size for optimal visual balance:

```tsx
// Small buttons
<GlassIconButton size="sm" aria-label="Small action">
  <Icon size={14} />
</GlassIconButton>

// Medium buttons (default)
<GlassIconButton size="md" aria-label="Medium action">
  <Icon size={16} />
</GlassIconButton>

// Large buttons
<GlassIconButton size="lg" aria-label="Large action">
  <Icon size={20} />
</GlassIconButton>
```

## Accessibility

### VoiceOver Support

The component provides comprehensive VoiceOver support:

```tsx
// Basic accessibility
<GlassIconButton aria-label="Add to favorites">
  <Heart size={20} />
</GlassIconButton>

// Dynamic labels for state changes
<GlassIconButton
  aria-label={isLiked ? 'Remove from favorites' : 'Add to favorites'}
  aria-pressed={isLiked}
  onClick={() => setIsLiked(!isLiked)}
>
  <Heart size={20} fill={isLiked ? 'currentColor' : 'none'} />
</GlassIconButton>

// Loading state
<GlassIconButton
  aria-label="Download file"
  aria-busy={isDownloading}
  disabled={isDownloading}
>
  <Download size={20} />
</GlassIconButton>
```

### Voice Control

Voice Control commands work automatically with proper labels:

```tsx
// User can say: "Tap Settings"
<GlassIconButton aria-label="Settings">
  <Settings size={20} />
</GlassIconButton>

// User can say: "Tap Share"
<GlassIconButton aria-label="Share">
  <Share size={20} />
</GlassIconButton>
```

### Keyboard Navigation

Full keyboard support is built-in:

- **Tab**: Navigate between buttons
- **Space/Enter**: Activate button
- **Escape**: Cancel action (in modals)

### Color Contrast

All variants meet WCAG AA contrast requirements:

- **Primary**: 4.5:1 contrast ratio
- **Secondary**: 4.5:1 contrast ratio
- **Tertiary**: 3:1 contrast ratio (non-text)
- **Ghost**: 4.5:1 contrast ratio
- **Destructive**: 4.5:1 contrast ratio

## Best Practices

### Do's ✅

- **Use descriptive aria-labels** for all icon buttons
- **Match icon size to button size** for visual balance
- **Group related actions** using consistent spacing
- **Provide immediate feedback** for state changes
- **Use standard icons** that users recognize
- **Maintain 44pt minimum touch targets**

```tsx
// ✅ Good - Clear label and appropriate size
<GlassIconButton aria-label='Add to favorites' size='md'>
  <Heart size={16} />
</GlassIconButton>
```

### Don'ts ❌

- **Don't use ambiguous icons** without clear context
- **Don't make touch targets too small**
- **Don't overuse destructive variants**
- **Don't forget loading states** for async actions
- **Don't use inconsistent icon styles**

```tsx
// ❌ Avoid - Unclear purpose and missing label
<GlassIconButton>
  <Circle size={16} />
</GlassIconButton>

// ❌ Avoid - Icon too small for button size
<GlassIconButton size="lg">
  <Heart size={12} />
</GlassIconButton>
```


### macOS

```tsx
// Keyboard shortcut integration
<GlassIconButton
  aria-label="Save (⌘S)"
  onClick={handleSave}
  onKeyDown={(e) => {
    if (e.metaKey && e.key === 's') {
      e.preventDefault()
      handleSave()
    }
  }}
>
  <Save size={20} />
</GlassIconButton>

// Tooltip for desktop
<Tooltip content="Save your changes">
  <GlassIconButton aria-label="Save">
    <Save size={20} />
  </GlassIconButton>
</Tooltip>
```
