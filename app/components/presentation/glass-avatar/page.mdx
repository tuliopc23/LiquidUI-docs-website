# GlassAvatar

User representation component using Apple's Liquid Glass design language, suitable for profiles, teams, and social contexts.

## Overview

GlassAvatars provide a flexible way to visually represent users or group members in an application. They support images, initials, or icons, and incorporate Liquid Glass effects to maintain elegant styling in all contexts.

## API Reference

### GlassAvatar Props

| Prop        | Type                                        | Default    | Description                       |
| ----------- | ------------------------------------------- | ---------- | --------------------------------- |
| `size`      | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'`      | `'md'`     | Avatar size                       |
| `image`     | `string`                                    | -          | URL to avatar image               |
| `initials`  | `string`                                    | -          | Initials for the avatar           |
| `icon`      | `string`                                    | -          | Icon name if no image or initials |
| `status`    | `'online' \| 'offline' \| 'busy' \| 'away'` | -          | Status indicator                  |
| `shape`     | `'circle' \| 'square' \| 'rounded'`         | `'circle'` | Avatar shape                      |
| `className` | `string`                                    | ''         | Additional CSS classes            |

## Accessibility

### Screen Reader Support

Ensure avatars provide proper labels for accessibility.

```jsx
<
 role="img" aria-label="User avatar for John Doe"
}
```

### Keyboard Navigation

Ensure avatars are accessible through keyboard navigation when interactive.

| Key             | Description                   |
| --------------- | ----------------------------- |
| `Tab`           | Navigate to focusable avatars |
| `Enter`/`Space` | Activate avatar action        |

## Best Practices

### Do's

- **Ensure diverse representations**: Use initials or icons for non-image scenarios
- **Provide status indicators**: Reflect user availability clearly
- **Optimize for performance**: Use appropriately sized image assets

### Don'ts

- **Don't rely on images alone**: Always provide fallbacks
- **Don't use low contrast combinations**: Ensure visibility

---

_GlassAvatar encapsulates user representation in a visually appealing and accessible manner, aligned with Apple's design principles._
