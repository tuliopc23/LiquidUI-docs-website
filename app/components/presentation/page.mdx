# Presentation

Components that display information and content to users with Apple's Liquid Glass design language.

## Overview

Presentation components in Liquidify provide sophisticated ways to display information, status, and content through elegant visual elements. Each component leverages Apple's Liquid Glass material system to create clear communication while maintaining beautiful aesthetics.

## Components

### [Glass Avatar](/components/presentation/glass-avatar)

User representation with elegant liquid glass styling.

- **Types**: image, initials, icon avatars
- **Sizes**: xs, sm, md, lg, xl
- **Features**: status indicators, fallbacks, groups
- **Use cases**: user profiles, team displays, comments

### [Glass Badge](/components/presentation/glass-badge)

Status indicators and labels with contextual styling.

- **Variants**: default, success, warning, error, info
- **Types**: dot, pill, counter badges
- **Features**: icons, animations, positioning
- **Use cases**: notifications, status, labels

### [Glass Chart](/components/presentation/glass-chart)

Data visualization with liquid glass aesthetics.

- **Types**: line, bar, pie, area charts
- **Features**: animations, tooltips, legends
- **Integration**: Chart.js, D3, custom data
- **Use cases**: dashboards, analytics, reports

### [Glass Loading](/components/presentation/glass-loading)

Loading states and progress indicators.

- **Types**: spinner, skeleton, progress bar
- **Variants**: determinate, indeterminate
- **Features**: smooth animations, accessibility
- **Use cases**: async operations, data loading

### [Glass Notification](/components/presentation/glass-notification)

User alerts and messages with contextual styling.

- **Types**: toast, banner, inline notifications
- **Variants**: success, warning, error, info
- **Features**: auto-dismiss, actions, positioning
- **Use cases**: feedback, alerts, confirmations

### [Glass Toast](/components/presentation/glass-toast)

Temporary messages with elegant animations.

- **Positioning**: top, bottom, corners
- **Features**: stacking, auto-dismiss, actions
- **Types**: simple, rich content
- **Use cases**: success messages, quick feedback

### [Glass Tooltip](/components/presentation/glass-tooltip)

Contextual help text with smart positioning.

- **Triggers**: hover, focus, click
- **Positioning**: all directions with auto-adjustment
- **Features**: arrows, delays, rich content
- **Use cases**: help text, definitions, shortcuts

## Design Principles

### Apple Human Interface Guidelines Compliance

#### Clarity

- **Clear information hierarchy**: Proper visual weight and contrast
- **Readable content**: Optimized typography and spacing
- **Purposeful visuals**: Every element serves a clear function

#### Deference

- **Content-focused**: Presentations support rather than dominate
- **Contextual styling**: Appropriate emphasis for content type
- **Subtle animations**: Enhance without distracting

#### Depth

- **Layered information**: Clear z-index and elevation hierarchy
- **Realistic materials**: Physics-based transparency and blur
- **Spatial relationships**: Logical component positioning

### Liquid Glass Integration

All presentation components utilize Liquidify's advanced material system:

- **Adaptive contrast**: Automatic adjustment for readability
- **Dynamic transparency**: Context-aware opacity levels
- **Intelligent blur**: Backdrop filtering that enhances content
- **Smooth transitions**: Seamless state changes and animations

### Accessibility Standards

#### Visual Accessibility

- **High contrast**: WCAG AA compliance for all text
- **Color independence**: Information not conveyed by color alone
- **Scalable typography**: Support for Dynamic Type and zoom
- **Reduced motion**: Respect for motion preferences

#### Screen Reader Support

- **Semantic markup**: Proper roles and labels
- **Live regions**: Dynamic content announcements
- **Descriptive text**: Clear purpose and current state
- **Logical structure**: Intuitive content hierarchy

#### Keyboard Navigation

- **Focus management**: Logical tab order and indicators
- **Standard shortcuts**: Consistent with platform conventions
- **Alternative access**: Multiple ways to access information

_Presentation components provide the visual foundation for communicating information effectively while maintaining Liquidify's beautiful and accessible design standards._
