# Glass Responsive Card

An adaptive card component with glassmorphism aesthetics, designed to respond to different screen sizes and layouts using Apple's Liquid Glass design language.

## Features

- **Responsive Design**: Adapts to different screen sizes and orientations
- **Glassmorphism Effects**: Translucent background with blur effects
- **Flexible Layout**: Automatic layout adjustments based on content
- **Breakpoint Awareness**: Customizable behavior at different breakpoints
- **Content Stacking**: Intelligent content reorganization on smaller screens
- **Accessibility**: Full keyboard navigation and screen reader support
- **Customizable**: Extensive theming and styling options

## Installation

```bash
npm install liquidify
```

## Basic Usage

```jsx
import { GlassResponsiveCard } from 'liquidify';

function App() {
  return (
    <GlassResponsiveCard
      title='Dashboard Stats'
      content='View your key metrics and performance indicators.'
      image='/dashboard-image.jpg'
      responsive={{
        mobile: { layout: 'vertical', padding: 'small' },
        tablet: { layout: 'horizontal', padding: 'medium' },
        desktop: { layout: 'horizontal', padding: 'large' },
      }}
    />
  );
}
```

## Integration with GSAP

The Glass Responsive Card component leverages GSAP for smooth animations:

```jsx
import { gsap } from 'gsap';

// Layout transition animation
const animateLayoutChange = (element, newLayout) => {
  gsap.to(element, {
    duration: 0.3,
    ease: 'power2.out',
    onComplete: () => {
      // Apply new layout
      element.setAttribute('data-layout', newLayout);
    },
  });
};

// Card entrance animation
const animateCardEntrance = element => {
  gsap.fromTo(
    element,
    { opacity: 0, y: 20 },
    {
      opacity: 1,
      y: 0,
      duration: 0.5,
      ease: 'power2.out',
    }
  );
};
```

### Conditional Rendering

```jsx
const ResponsiveCard = ({ data, userRole }) => {
  const getResponsiveConfig = () => {
    const baseConfig = {
      mobile: { layout: 'vertical', padding: 'small' },
      desktop: { layout: 'horizontal', padding: 'large' },
    };

    if (userRole === 'admin') {
      return {
        ...baseConfig,
        mobile: { ...baseConfig.mobile, showMetric: true },
        desktop: { ...baseConfig.desktop, showMetric: true },
      };
    }

    return baseConfig;
  };

  return (
    <GlassResponsiveCard
      title={data.title}
      content={data.content}
      metric={userRole === 'admin' ? data.metric : undefined}
      responsive={getResponsiveConfig()}
    />
  );
};
```

### Container Query Support

```jsx
const useContainerQuery = ref => {
  const [containerSize, setContainerSize] = useState('large');

  useEffect(() => {
    const observer = new ResizeObserver(entries => {
      const { width } = entries[0].contentRect;
      if (width < 300) setContainerSize('small');
      else if (width < 600) setContainerSize('medium');
      else setContainerSize('large');
    });

    if (ref.current) {
      observer.observe(ref.current);
    }

    return () => observer.disconnect();
  }, [ref]);

  return containerSize;
};

const ContainerAwareCard = ({ children }) => {
  const containerRef = useRef();
  const containerSize = useContainerQuery(containerRef);

  return (
    <div ref={containerRef}>
      <GlassResponsiveCard
        title='Container Aware'
        content='This card adapts to its container size'
        responsive={{
          [containerSize]: {
            layout: containerSize === 'small' ? 'compact' : 'default',
            padding: containerSize === 'small' ? 'small' : 'medium',
          },
        }}
      />
    </div>
  );
};
```
