# Glass Responsive Button

An adaptive button component with glassmorphism aesthetics, designed to respond to different screen sizes and contexts using Apple's Liquid Glass design language.

## Features

- **Responsive Design**: Adapts to different screen sizes and orientations
- **Glassmorphism Effects**: Translucent background with blur effects
- **Adaptive Layout**: Changes appearance based on available space
- **Touch Optimized**: Enhanced touch targets for mobile devices
- **Breakpoint Awareness**: Customizable behavior at different breakpoints
- **Accessibility**: Full keyboard navigation and screen reader support
- **Customizable**: Extensive theming and styling options

## Installation

```bash
npm install liquidify
```

## Basic Usage

```jsx
import { GlassResponsiveButton } from 'liquidify'

function App() {
  return (
    <GlassResponsiveButton
      icon={<PlusIcon className="w-5 h-5" />}
      label="Add Item"
      onClick={() => console.log('Button clicked')}
      responsive={{
        mobile: { showLabel: false },
        tablet: { showLabel: true },
        desktop: { showLabel: true }
      }}
    />
  )
}
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `React.ReactNode` | - | Button icon |
| `label` | `string` | - | Button label |
| `onClick` | `() => void` | - | Click handler |
| `variant` | `'primary' \| 'secondary' \| 'outline' \| 'danger' \| 'icon'` | `'primary'` | Button variant |
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | Button size |
| `responsive` | `ResponsiveConfig` | - | Responsive configuration |
| `floating` | `boolean` | `false` | Floating action button |
| `disabled` | `boolean` | `false` | Disabled state |
| `loading` | `boolean` | `false` | Loading state |
| `className` | `string` | - | Additional CSS classes |
| `style` | `CSSProperties` | - | Inline styles |

### ResponsiveConfig Interface

```typescript
interface ResponsiveConfig {
  mobile?: {
    showLabel?: boolean
    size?: 'small' | 'medium' | 'large'
    variant?: 'primary' | 'secondary' | 'outline' | 'danger' | 'icon'
    position?: 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right'
  }
  tablet?: {
    showLabel?: boolean
    size?: 'small' | 'medium' | 'large'
    variant?: 'primary' | 'secondary' | 'outline' | 'danger' | 'icon'
    position?: 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right'
  }
  desktop?: {
    showLabel?: boolean
    size?: 'small' | 'medium' | 'large'
    variant?: 'primary' | 'secondary' | 'outline' | 'danger' | 'icon'
    position?: 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right'
  }
}
```

## Integration with GSAP

The Glass Responsive Button component leverages GSAP for smooth animations:

```jsx
import { gsap } from 'gsap'

// Responsive transition animation
const animateResponsiveChange = (element, newProps) => {
  gsap.to(element, {
    ...newProps,
    duration: 0.3,
    ease: 'power2.out'
  })
}

// Floating button entrance
const animateFloatingButton = (element) => {
  gsap.fromTo(element,
    { scale: 0, opacity: 0 },
    {
      scale: 1,
      opacity: 1,
      duration: 0.4,
      ease: 'back.out(1.7)'
    }
  )
}
```

### Context-Aware Adaptation

```jsx
const useResponsiveButton = (containerWidth) => {
  const [config, setConfig] = useState({})

  useEffect(() => {
    if (containerWidth < 200) {
      setConfig({ showLabel: false, size: 'small' })
    } else if (containerWidth < 400) {
      setConfig({ showLabel: true, size: 'medium' })
    } else {
      setConfig({ showLabel: true, size: 'large' })
    }
  }, [containerWidth])

  return config
}

const ResponsiveToolbar = () => {
  const containerRef = useRef()
  const [containerWidth, setContainerWidth] = useState(0)
  const buttonConfig = useResponsiveButton(containerWidth)

  return (
    <div ref={containerRef} className="flex space-x-2">
      <GlassResponsiveButton
        icon={<PlusIcon className="w-5 h-5" />}
        label="Add"
        onClick={() => handleAdd()}
        {...buttonConfig}
      />
    </div>
  )
}
```
