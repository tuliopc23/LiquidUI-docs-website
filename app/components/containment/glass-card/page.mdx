# Glass Card

Flexible container components using Apple's Liquid Glass material to group related content with elegant visual hierarchy.

## Overview

Glass cards provide a sophisticated way to organize and present related information using Apple's Liquid Glass design language. They create visual groupings while maintaining the content's prominence through intelligent transparency and backdrop filtering that adapts to environmental context.

## Anatomy

The Glass Card consists of:
- **Container**: Liquid glass surface with backdrop blur
- **Header** (optional): Title and metadata area
- **Content Area**: Main content with proper padding
- **Footer** (optional): Actions or additional information
- **Border**: Subtle liquid glass border for definition

## States

### Default State
Clean liquid glass appearance with backdrop blur and subtle border definition.

### Interactive States
- **Hover**: Enhanced glass clarity with gentle elevation
- **Focus**: Accessibility focus outline when card is interactive
- **Pressed**: Subtle compression for interactive cards
- **Elevated**: Increased blur and shadow for overlays

### Responsive Behavior
- **Mobile**: Full-width with appropriate margins
- **Tablet**: Flexible grid layouts
- **Desktop**: Optimized spacing and proportions

## Usage Guidelines

### When to Use
- Grouping related content logically
- Creating visual hierarchy in layouts
- Presenting information cards or tiles
- Building dashboard components
- Product or feature showcases

### When Not to Use
- Single pieces of content that don't need grouping
- When content transparency would reduce readability
- Overly complex nested card structures

## Accessibility

### Voice Control
Cards with interactive elements support Voice Control navigation.

### Keyboard Navigation
- **Tab**: Navigate through interactive elements within card
- **Enter/Space**: Activate card if clickable
- **Arrow Keys**: Navigate between cards in grid layouts

### Screen Reader Support
- Proper semantic structure with headings
- Clear content organization
- Descriptive labels for interactive cards

### High Contrast Mode
Maintains proper contrast and border visibility.

### Reduced Motion
Respects motion preferences for hover and interaction animations.

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'subtle' \| 'prominent' \| 'overlay'` | `'default'` | Visual prominence level |
| `padding` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Internal spacing |
| `radius` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'lg'` | Corner radius following Apple guidelines |
| `shadow` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Elevation shadow |
| `interactive` | `boolean` | `false` | Enables hover and click states |
| `header` | `ReactNode` | `undefined` | Optional header content |
| `footer` | `ReactNode` | `undefined` | Optional footer content |
| `children` | `ReactNode` | - | Main card content |
| `onClick` | `(event: MouseEvent) => void` | - | Click handler for interactive cards |

### Variants

- **Default**: Standard liquid glass with medium opacity
- **Subtle**: Lighter glass for secondary content
- **Prominent**: Enhanced glass for important content
- **Overlay**: Maximum opacity for modal-like content

## Design Tokens

### Colors
- **Default**: 15% white/black opacity with backdrop blur
- **Subtle**: 8% white/black opacity with light blur
- **Prominent**: 25% white/black opacity with enhanced blur
- **Border**: 1px solid with 18% white/black opacity

### Typography
Cards should use proper Apple HIG typography hierarchy:
- **Titles**: `hig-headline` or `hig-title-3`
- **Body**: `hig-body` for main content
- **Metadata**: `hig-caption-1` for secondary information

### Spacing
- **Small**: 12pt padding
- **Medium**: 16pt padding (default)
- **Large**: 24pt padding
- **XLarge**: 32pt padding

### Elevation
- **None**: No shadow
- **Small**: 2pt blur, 1pt offset
- **Medium**: 8pt blur, 4pt offset (default)
- **Large**: 16pt blur, 8pt offset

### Motion
- **Hover**: `translateY(-2px)` over 200ms
- **Press**: `scale(0.98)` over 100ms
- **Glass Enhancement**: Opacity increase over 150ms
