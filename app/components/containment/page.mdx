# Containment

Components that group and organize related content using Apple's Liquid Glass design language.

## Overview

Containment components in Liquidify provide sophisticated ways to organize and present content through elegant visual groupings. Each component leverages Apple's Liquid Glass material system to create clear hierarchy while maintaining content prominence through intelligent transparency and adaptive blur effects.

## Components

### [Glass Card](/components/containment/glass-card)

Flexible container components for grouping related content with visual hierarchy.

- **Variants**: default, subtle, prominent, overlay
- **States**: interactive, elevated, focused
- **Features**: headers, footers, responsive layouts
- **Use cases**: dashboards, product cards, feature showcases

### [Glass Modal](/components/containment/glass-modal)

Full-screen overlay dialogs for focused user interactions.

- **Types**: dialog, sheet, fullscreen
- **Features**: backdrop blur, focus management, escape handling
- **Accessibility**: keyboard navigation, screen reader support
- **Use cases**: forms, confirmations, detailed views

### [Glass Popover](/components/containment/glass-popover)

Contextual overlays for supplementary information and actions.

- **Positioning**: smart placement, collision detection
- **Triggers**: click, hover, focus
- **Features**: arrow indicators, responsive sizing
- **Use cases**: tooltips, menus, form helpers

### [Glass Command](/components/containment/glass-command)

Command palette interface for quick actions and navigation.

- **Features**: fuzzy search, keyboard shortcuts, categories
- **Navigation**: arrow keys, enter selection
- **Extensibility**: custom commands, plugins
- **Use cases**: power user interfaces, search, navigation

## Design Principles

### Apple Human Interface Guidelines Compliance

#### Clarity

- **Clear boundaries**: Well-defined content groupings
- **Visual hierarchy**: Consistent spacing and typography
- **Content focus**: Container supports content without distraction

#### Deference

- **Adaptive transparency**: Background content remains visible
- **Subtle borders**: Definition without overwhelming
- **Content-first**: Container enhances rather than dominates

#### Depth

- **Layered organization**: Clear spatial relationships
- **Contextual elevation**: Appropriate z-index management
- **Realistic materials**: Physics-based blur and transparency

### Liquid Glass Integration

All containment components utilize Liquidify's advanced material system:

- **Dynamic blur**: Adaptive backdrop filtering based on content
- **Intelligent opacity**: Context-aware transparency levels
- **Smooth transitions**: Seamless state changes and animations
- **Environmental adaptation**: Responds to light and dark themes

### Accessibility Standards

#### Focus Management

- **Logical tab order**: Intuitive navigation within containers
- **Focus containment**: Modal dialogs trap focus appropriately
- **Focus restoration**: Return focus to trigger elements

#### Screen Reader Support

- **Semantic structure**: Proper heading hierarchy and landmarks
- **Clear labels**: Descriptive container purposes
- **Live regions**: Dynamic content announcements

#### Keyboard Navigation

- **Standard shortcuts**: Escape to close, arrow navigation
- **Custom controls**: Command palette keyboard shortcuts
- **Alternative access**: Multiple ways to access content

---

_Containment components provide the foundational structure for organizing content in Liquidify applications, following Apple's design principles for clarity, hierarchy, and accessibility._
