# Components

Comprehensive collection of React components built with Apple's Liquid Glass design language and Human Interface Guidelines.

## Overview

Liquidify provides a complete set of UI components that implement Apple's newest design paradigm: Liquid Glass. Each component follows strict accessibility standards, supports all Apple platforms, and provides delightful user interactions through sophisticated material properties.

## Component Categories

## Design Principles

### Liquid Glass Material

All components use Apple's Liquid Glass material, which:

- Adapts to surrounding content intelligently
- Provides consistent visual hierarchy
- Maintains accessibility in all lighting conditions
- Supports both light and dark environments seamlessly

### Apple Human Interface Guidelines

Every component strictly follows Apple HIG principles:

- **Clarity**: Clear visual communication
- **Deference**: Content over chrome
- **Depth**: Realistic layering and motion

### Accessibility First

Built with comprehensive accessibility support:

- **VoiceOver**: Full screen reader compatibility
- **Voice Control**: Complete voice navigation
- **Keyboard Navigation**: Full keyboard accessibility
- **High Contrast**: Enhanced visibility options
- **Reduced Motion**: Respects user preferences

### Performance Optimized

- Tree-shakable imports for minimal bundle size
- Optimized rendering with React 19
- Efficient animation with GSAP
- Lazy loading for complex components

## Getting Started

### Installation

```bash
npm install liquidify
```

### Basic Usage

```tsx
import { GlassButton, GlassCard } from 'liquidify';

export default function App() {
  return (
    <GlassCard>
      <h1 className='hig-title-1 mb-4'>Welcome to Liquidify</h1>
      <p className='hig-body mb-6'>
        Experience the future of interface design with Liquid Glass.
      </p>
      <GlassButton variant='primary'>Get Started</GlassButton>
    </GlassCard>
  );
}
```

### Theme Configuration

```tsx
import { ThemeProvider } from 'liquidify';

export default function RootLayout({ children }) {
  return <ThemeProvider>{children}</ThemeProvider>;
}
```

## Design Tokens

Liquidify uses Apple's design token system:

- **Colors**: Official Apple system colors
- **Typography**: SF Pro Display and SF Pro Text fonts
- **Spacing**: 4pt grid system
- **Elevation**: Consistent shadow system
- **Motion**: Apple spring animations
