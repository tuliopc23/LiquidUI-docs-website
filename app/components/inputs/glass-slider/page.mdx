# Glass Slider

A component to select a value from a defined range with a sliding interface, offering both discrete and continuous value options.

## Overview

The `GlassSlider` component provides a smooth and interactive way for users to select a value from a defined range, with customization options for both style and functionality.

## Basic Usage

```tsx
import { GlassSlider } from 'liquidify';

function App() {
  const [value, setValue] = useState(50);

  return (
    <GlassSlider value={value} onChange={setValue} min={0} max={100} step={1} />
  );
}
```

## Interactive Demo

```tsx
import { useState } from 'react';
import { GlassSlider } from 'liquidify';

function SliderDemo() {
  const [volume, setVolume] = useState(50);
  const [brightness, setBrightness] = useState(75);

  return (
    <div className='space-y-6 p-6'>
      {/* Volume Slider */}
      <section>
        <h3 className='hig-title-3 mb-4'>Volume</h3>
        <GlassSlider
          value={volume}
          onChange={setVolume}
          min={0}
          max={100}
          step={1}
          tooltipVisible
          label='Volume'
        />
        <p className='text-sm mt-2'>Current Volume: {volume}%</p>
      </section>

      {/* Brightness Slider */}
      <section>
        <h3 className='hig-title-3 mb-4'>Brightness</h3>
        <GlassSlider
          value={brightness}
          onChange={setBrightness}
          min={0}
          max={100}
          step={5}
          tooltipVisible
          label='Brightness'
        />
        <p className='text-sm mt-2'>Current Brightness: {brightness}%</p>
      </section>
    </div>
  );
}
```

## API Reference

### Props

| Prop             | Type                      | Default | Description                                       |
| ---------------- | ------------------------- | ------- | ------------------------------------------------- |
| `value`          | `number`                  | `0`     | Current value of the slider                       |
| `onChange`       | `(value: number) => void` | -       | Callback when the slider value changes            |
| `min`            | `number`                  | `0`     | Minimum value allowed                             |
| `max`            | `number`                  | `100`   | Maximum value allowed                             |
| `step`           | `number`                  | `1`     | Interval between selectable values in the slider  |
| `disabled`       | `boolean`                 | `false` | Disables the slider                               |
| `tooltipVisible` | `boolean`                 | `false` | Shows current value in a tooltip above the slider |
| `label`          | `string`                  | ''      | Label text for the slider                         |

## Features

### Discrete vs Continuous

- **Discrete Mode**: Allow only fixed increments

```tsx
<GlassSlider min={0} max={10} step={1} />
```

- **Continuous Mode**: Freeform slider within a range

```tsx
<GlassSlider min={0} max={100} step={0.1} />
```

### Custom Styling

Style the track and thumb to fit your design needs

```tsx
<GlassSlider
  trackClassName='bg-glass-secondary'
  thumbClassName='bg-glass-primary'
/>
```

## Accessibility

- **Keyboard Navigation**: Arrow keys to adjust values
- **ARIA support** for better screen reader compatibility

### Keyboard Example

```tsx
<GlassSlider aria-label='Adjust volume' value={volume} onChange={setVolume} />
```

## Best Practices

### Do's ✅

- **Use clear labels** to indicate slider function
- **Define appropriate min/max** range for context
- **Show tooltip** for precise value adjustments

### Don'ts ❌

- **Avoid overly large ranges** without context
- **Don't rely solely on visual cues** for slider interaction
