# Glass Radio

A component for selecting one option from a group of choices, ensuring mutually exclusive selections with accessible keyboard navigation.

## Overview

The `GlassRadio` component allows users to select a single option from a closely related group, supporting both horizontal and vertical layouts. It's ideal for demographic data and settings.

## Basic Usage

```tsx
import { GlassRadioGroup, GlassRadioItem } from 'liquidify';

function App() {
  const [choice, setChoice] = useState('');

  return (
    <GlassRadioGroup value={choice} onValueChange={setChoice}>
      <GlassRadioItem value='apple'>Apple</GlassRadioItem>
      <GlassRadioItem value='banana'>Banana</GlassRadioItem>
      <GlassRadioItem value='orange'>Orange</GlassRadioItem>
    </GlassRadioGroup>
  );
}
```

## Advanced Example

```tsx
import { useState } from 'react';
import { GlassRadioGroup, GlassRadioItem } from 'liquidify';

function RadioDemo() {
  const [color, setColor] = useState('');

  const colors = [
    { value: 'red', label: 'Red' },
    { value: 'green', label: 'Green' },
    { value: 'blue', label: 'Blue' },
  ];

  return (
    <div className='space-y-6 p-6'>
      <section>
        <h3 className='hig-title-3 mb-4'>Favorite Color</h3>
        <GlassRadioGroup value={color} onValueChange={setColor}>
          {colors.map(color => (
            <GlassRadioItem key={color.value} value={color.value}>
              {color.label}
            </GlassRadioItem>
          ))}
        </GlassRadioGroup>
        <p className='text-sm mt-2'>Selected: {color}</p>
      </section>
    </div>
  );
}
```

## API Reference

### GlassRadioGroup Props

| Prop            | Type                      | Default | Description                    |
| --------------- | ------------------------- | ------- | ------------------------------ |
| `value`         | `string`                  | `''`    | Currently selected value       |
| `onValueChange` | `(value: string) => void` | -       | Handler for value changes      |
| `name`          | `string`                  | -       | Group name for form submission |
| `disabled`      | `boolean`                 | `false` | Disables the entire group      |

### GlassRadioItem Props

| Prop       | Type        | Default | Description                |
| ---------- | ----------- | ------- | -------------------------- |
| `value`    | `string`    | -       | Unique value for this item |
| `disabled` | `boolean`   | `false` | Disables this item         |
| `children` | `ReactNode` | -       | Label content              |

## Accessibility

- **ARIA Support**: Proper `role="radiogroup"` and `role="radio"`
- **Keyboard Navigation**: Arrow keys, Space, Tab
- **Screen Reader**: Clear labels and state announcements

## Best Practices

- Use for mutually exclusive choices
- Provide clear, descriptive labels
- Group related options together
- Consider using a select dropdown for many options
