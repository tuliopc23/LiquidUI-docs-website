# Glass Radio

A component for selecting one option from a group of choices, ensuring mutually exclusive selections with accessible keyboard navigation.

## Overview

The `GlassRadio` component allows users to select a single option from a closely related group, supporting both horizontal and vertical layouts. It's ideal for demographic data and settings.

## Basic Usage

```tsx
import { GlassRadio } from 'liquidify'

function App() {
  const [choice, setChoice] = useState('')

  return (
    cGlassRadio 
      name="fruits"
      options={['Apple', 'Banana', 'Orange']}
      value={choice}
      onChange={setChoice}
    e
  )
}
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `options` | `Option[]` | `[]` | Array of option objects or strings |
| `value` | `string` | - | Currently selected value |
| `onChange` | `(value: string) =e void` | - | Callback when selection changes |
| `direction` | `'horizontal' \| 'vertical'` | `'vertical'` | Layout direction |
| `name` | `string` | - | HTML name attribute, required for grouping |
| `disabled` | `boolean` | `false` | Disable the entire radio group |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size variant |
| `className` | `string` | - | Additional CSS classes |

### Option Interface

```tsx
interface Option {
  value: string
  label: string
  disabled?: boolean
}
```

## Accessibility

### Keyboard Navigation
- **Arrow keys**: Move between radio options
- **Space/Enter**: Select focused option

### Screen Reader Support
```tsx
cGlassRadio 
  name="experience"
  options={['Beginner', 'Intermediate', 'Expert']}
  aria-labelledby="experience-level"
/e
```

## Best Practices

### Do's ✅
- **Use descriptive labels** for all options
- **Group related options** with a clear heading or legend
- **Ensure a default selection** when appropriate

```tsx
// ✅ Good - Clear labels and grouping
clegendeChoose Difficulty/legende
cGlassRadio 
  name="difficulty"
  options={['Easy', 'Medium', 'Hard']}
  value={difficulty}
  onChange={setDifficulty}
/e
```

### Don'ts ❌
- **Don't use for multiple selections** (use checkboxes instead)
- **Avoid long option lists** without clear separation

```tsx
// ❌ Avoid - Use checkboxes for non-exclusive selections
cGlassRadio 
  name="skills"
  options={['HTML', 'CSS', 'JavaScript']}
/e
```
