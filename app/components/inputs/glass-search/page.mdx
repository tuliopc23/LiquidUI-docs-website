# Glass Search

A modern search input component with glassmorphism aesthetics, perfect for implementing search functionality with Apple's Liquid Glass design language.

## Features

- **Glassmorphism Design**: Translucent background with blur effects
- **Search Icon Integration**: Built-in search icon with customizable placement
- **Real-time Search**: Instant feedback and suggestions
- **Responsive Design**: Adapts to different screen sizes
- **Accessibility**: Full keyboard navigation and screen reader support
- **Customizable**: Extensive theming and styling options

## Installation

```bash
npm install liquidify
```

## Basic Usage

```jsx
import { GlassSearch } from '@/components/MDXLiquidifyComponents';

function App() {
  const [searchValue, setSearchValue] = useState('');

  return (
    <GlassSearch
      value={searchValue}
      onChange={value => setSearchValue(value)}
      placeholder='Search components...'
    />
  );
}
```

## API Reference

### Props

| Prop          | Type                      | Default                   | Description                                 |
| ------------- | ------------------------- | ------------------------- | ------------------------------------------- |
| `value`       | `string`                  | `""`                      | The search input value                      |
| `onChange`    | `(value: string) => void` | -                         | Callback fired when value changes           |
| `placeholder` | `string`                  | `"Search..."`             | Placeholder text                            |
| `icon`        | `React.ReactNode`         | `<MagnifyingGlassIcon />` | Icon to display in the search input         |
| `suggestions` | `string[]`                | `[]`                      | Array of suggestion items                   |
| `loading`     | `boolean`                 | `false`                   | Show loading state                          |
| `disabled`    | `boolean`                 | `false`                   | Disable the search input                    |
| `clearable`   | `boolean`                 | `true`                    | Show clear button when input has value      |
| `onClear`     | `() => void`              | -                         | Callback fired when clear button is clicked |
| `onFocus`     | `() => void`              | -                         | Callback fired when input is focused        |
| `onBlur`      | `() => void`              | -                         | Callback fired when input loses focus       |
| `className`   | `string`                  | -                         | Additional CSS classes                      |
| `style`       | `CSSProperties`           | -                         | Inline styles                               |
