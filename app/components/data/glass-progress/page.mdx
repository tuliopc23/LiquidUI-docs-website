# GlassProgress

Progress indicators for loading states and task completion using Apple's Liquid Glass design language.

## Overview

GlassProgress provides visual feedback for ongoing processes, loading states, and task completion. It offers multiple variants to match different use cases while maintaining accessibility and smooth animations.

## Basic Usage

```tsx
import { GlassProgress } from '@liquidify/ui';

function BasicProgress() {
  return <GlassProgress value={65} />;
}
```

## API Reference

### GlassProgress Props

| Prop            | Type                                             | Default     | Description                 |
| --------------- | ------------------------------------------------ | ----------- | --------------------------- |
| `value`         | `number`                                         | -           | Progress value (0-100)      |
| `type`          | `'linear' \| 'circular'`                         | `'linear'`  | Progress indicator type     |
| `variant`       | `'default' \| 'success' \| 'warning' \| 'error'` | `'default'` | Color variant               |
| `size`          | `'sm' \| 'md' \| 'lg'`                           | `'md'`      | Component size              |
| `indeterminate` | `boolean`                                        | `false`     | Show indeterminate progress |
| `showLabel`     | `boolean`                                        | `false`     | Display progress percentage |
| `label`         | `string`                                         | -           | Custom label text           |

## Common Use Cases

### File Upload

```tsx
function FileUpload() {
  const [progress, setProgress] = useState(0);

  return (
    <div>
      <GlassProgress
        value={progress}
        label={`Uploading... ${progress}%`}
        showLabel
      />
    </div>
  );
}
```

### Loading States

```tsx
<GlassProgress indeterminate label='Loading content...' />
```

### Multi-step Process

```tsx
function StepProgress({ currentStep, totalSteps }) {
  const progress = (currentStep / totalSteps) * 100;

  return (
    <GlassProgress
      value={progress}
      label={`Step ${currentStep} of ${totalSteps}`}
      showLabel
    />
  );
}
```

## Accessibility

### ARIA Support

- **`role="progressbar"`**: Proper semantic meaning
- **`aria-valuenow`**: Current progress value
- **`aria-valuemin`** and **`aria-valuemax`**: Progress range
- **`aria-label`**: Descriptive progress label

### Screen Reader Support

```tsx
<GlassProgress value={75} aria-label='Upload progress: 75% complete' />
```
