# Glass Mobile Nav

A responsive mobile navigation component with glassmorphism aesthetics, designed for mobile-first experiences using Apple's Liquid Glass design language.

## Features

- **Mobile-First Design**: Optimized for touch interactions and small screens
- **Glassmorphism Effects**: Translucent background with blur effects
- **Gesture Support**: Swipe gestures for navigation
- **Bottom Navigation**: iOS-style bottom navigation bar
- **Responsive**: Adapts to different screen sizes and orientations
- **Accessibility**: Full touch and screen reader support
- **Customizable**: Extensive theming and styling options

## Installation

```bash
npm install liquidify
```

## Basic Usage

```jsx
import { GlassMobileNav } from 'liquidify';

function App() {
  const [activeTab, setActiveTab] = useState('home');

  const navItems = [
    { id: 'home', label: 'Home', icon: HomeIcon },
    { id: 'profile', label: 'Profile', icon: UserIcon },
    { id: 'notifications', label: 'Notifications', icon: BellIcon },
    { id: 'settings', label: 'Settings', icon: CogIcon },
  ];

  return (
    <GlassMobileNav
      items={navItems}
      activeTab={activeTab}
      onTabChange={setActiveTab}
    />
  );
}
```

## API Reference

### Props

| Prop          | Type                      | Default    | Description                     |
| ------------- | ------------------------- | ---------- | ------------------------------- |
| `items`       | `NavItem[]`               | `[]`       | Array of navigation items       |
| `activeTab`   | `string`                  | -          | Currently active tab ID         |
| `onTabChange` | `(tabId: string) => void` | -          | Callback fired when tab changes |
| `compact`     | `boolean`                 | `false`    | Hide labels and show icons only |
| `floating`    | `boolean`                 | `false`    | Floating navigation bar style   |
| `position`    | `'bottom' \| 'top'`       | `'bottom'` | Position of the navigation bar  |
| `disabled`    | `boolean`                 | `false`    | Disable all navigation items    |
| `className`   | `string`                  | -          | Additional CSS classes          |
| `style`       | `CSSProperties`           | -          | Inline styles                   |

### NavItem Interface

```typescript
interface NavItem {
  id: string;
  label?: string;
  icon: React.ComponentType<any>;
  badge?: number | string;
  disabled?: boolean;
  href?: string;
  onClick?: () => void;
}
```

## Accessibility

- **Touch Targets**: Minimum 44px touch targets for better usability
- **Screen Reader**: Properly labeled with ARIA attributes
- **Focus Management**: Clear focus indicators and logical tab order
- **Keyboard Support**: Full keyboard navigation support
- **Badge Announcements**: Screen reader support for notification badges

## Best Practices

1. **Touch Targets**: Ensure touch targets are at least 44px for mobile usability
2. **Icon Clarity**: Use clear, recognizable icons for navigation items
3. **Badge Usage**: Use badges sparingly for important notifications only
4. **Consistent Placement**: Keep navigation in consistent position across screens
5. **Performance**: Optimize for smooth animations and transitions

## Integration with GSAP

The Glass Mobile Nav component leverages GSAP for smooth animations:

```jsx
import { gsap } from 'gsap';

// Custom tab switching animation
const handleTabChange = tabId => {
  gsap.to('.glass-mobile-nav-indicator', {
    x: getTabPosition(tabId),
    duration: 0.3,
    ease: 'power2.out',
  });
};
```

## Gesture Support

The component supports swipe gestures for navigation:

```jsx
<GlassMobileNav
  items={navItems}
  activeTab={activeTab}
  onTabChange={setActiveTab}
  gestures={{
    swipeLeft: () => setActiveTab(getNextTab()),
    swipeRight: () => setActiveTab(getPrevTab()),
  }}
/>
```
