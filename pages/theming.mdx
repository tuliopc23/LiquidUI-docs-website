# Theming

Glass UI provides a powerful and flexible theming system that allows you to customize every aspect of your components to match your brand and design requirements.

## Design Tokens

Glass UI uses CSS custom properties (CSS variables) as design tokens, making it easy to customize colors, spacing, typography, and effects throughout your application.

### Color System

The color system is built around semantic color tokens that adapt to both light and dark themes:

```css
:root {
  /* Primary Colors */
  --glass-primary: #3b82f6;
  --glass-primary-hover: #2563eb;
  --glass-primary-text: #ffffff;
  
  /* Secondary Colors */
  --glass-secondary: #6366f1;
  --glass-secondary-hover: #4f46e5;
  --glass-secondary-text: #ffffff;
  
  /* Glass Effects */
  --glass-background: rgba(255, 255, 255, 0.1);
  --glass-background-hover: rgba(255, 255, 255, 0.2);
  --glass-border: rgba(255, 255, 255, 0.2);
  --glass-border-hover: rgba(255, 255, 255, 0.3);
  --glass-blur: 10px;
  --glass-shadow: 0 8px 32px rgba(31, 38, 135, 0.37);
  
  /* Semantic Colors */
  --glass-success: #10b981;
  --glass-warning: #f59e0b;
  --glass-error: #ef4444;
  --glass-info: #3b82f6;
  
  /* Text Colors */
  --glass-text-primary: #1f2937;
  --glass-text-secondary: #6b7280;
  --glass-text-muted: #9ca3af;
}
```

### Dark Theme

Dark theme colors are automatically applied when the `dark` class is present:

```css
.dark {
  /* Glass Effects for Dark Mode */
  --glass-background: rgba(0, 0, 0, 0.1);
  --glass-background-hover: rgba(0, 0, 0, 0.2);
  --glass-border: rgba(255, 255, 255, 0.1);
  --glass-border-hover: rgba(255, 255, 255, 0.2);
  
  /* Text Colors for Dark Mode */
  --glass-text-primary: #f9fafb;
  --glass-text-secondary: #d1d5db;
  --glass-text-muted: #9ca3af;
}
```

### Method 2: Theme Configuration Object

For more advanced theming, you can create a theme configuration object:

```tsx
// theme.config.ts
export const customTheme = {
  colors: {
    primary: {
      50: '#faf5ff',
      100: '#f3e8ff',
      500: '#8b5cf6',
      600: '#7c3aed',
      700: '#6d28d9',
    },
    glass: {
      background: 'rgba(139, 92, 246, 0.1)',
      backgroundHover: 'rgba(139, 92, 246, 0.2)',
      border: 'rgba(139, 92, 246, 0.2)',
      borderHover: 'rgba(139, 92, 246, 0.3)',
      blur: '12px',
    }
  },
  spacing: {
    xs: '0.25rem',
    sm: '0.5rem',
    md: '1rem',
    lg: '1.5rem',
    xl: '2rem',
  },
  borderRadius: {
    sm: '0.375rem',
    md: '0.5rem',
    lg: '0.75rem',
    xl: '1rem',
  }
}

// Apply theme
import { applyTheme } from 'glass-ui/theme'
applyTheme(customTheme)
```

## Responsive Theming

Glass UI supports responsive theming using CSS media queries:

```css
/* Mobile-first responsive theming */
:root {
  --glass-blur: 8px;
  --glass-spacing-md: 0.75rem;
}

@media (min-width: 768px) {
  :root {
    --glass-blur: 10px;
    --glass-spacing-md: 1rem;
  }
}

@media (min-width: 1024px) {
  :root {
    --glass-blur: 12px;
    --glass-spacing-md: 1.25rem;
  }
}
```

## Advanced Theming

### Dynamic Theme Switching

Implement dynamic theme switching with JavaScript:

```tsx
import { useState, useEffect } from 'react'

function ThemeProvider({ children }) {
  const [theme, setTheme] = useState('default')
  
  useEffect(() => {
    document.documentElement.className = `theme-${theme}`
  }, [theme])
  
  const themes = [
    { name: 'default', label: 'Default' },
    { name: 'ocean', label: 'Ocean' },
    { name: 'forest', label: 'Forest' },
    { name: 'sunset', label: 'Sunset' },
  ]
  
  return (
    <div>
      <div className="theme-selector">
        {themes.map((t) => (
          <button
            key={t.name}
            onClick={() => setTheme(t.name)}
            className={theme === t.name ? 'active' : ''}
          >
            {t.label}
          </button>
        ))}
      </div>
      {children}
    </div>
  )
}
```

## Accessibility Considerations

When customizing themes, ensure accessibility standards are maintained:

### Color Contrast

```css
/* Ensure sufficient contrast ratios */
:root {
  --glass-text-primary: #1f2937; /* 4.5:1 contrast ratio */
  --glass-text-secondary: #4b5563; /* 3:1 contrast ratio */
}

.dark {
  --glass-text-primary: #f9fafb; /* 4.5:1 contrast ratio */
  --glass-text-secondary: #d1d5db; /* 3:1 contrast ratio */
}
```

### Focus Indicators

```css
/* Maintain visible focus indicators */
.glass-button:focus {
  outline: 2px solid var(--glass-primary);
  outline-offset: 2px;
}

/* High contrast mode support */
@media (prefers-contrast: high) {
  .glass-effect {
    background: white !important;
    border: 2px solid black !important;
    backdrop-filter: none !important;
  }
}
```

### Reduced Motion

```css
/* Respect user motion preferences */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

## Best Practices

1. **Start with design tokens** - Use CSS variables for consistency
2. **Test in both themes** - Ensure your customizations work in light and dark modes
3. **Maintain contrast ratios** - Follow WCAG guidelines for accessibility
4. **Use semantic naming** - Name your custom properties based on purpose, not appearance
5. **Document your theme** - Keep a style guide for your custom theme
6. **Test across devices** - Ensure glass effects work on different screen types

<Callout type="warning">
  Glass effects may not be supported on older browsers. Always provide fallback styles for better compatibility.
</Callout>

## Troubleshooting

### Common Issues

1. **Glass effects not showing**: Ensure `backdrop-filter` is supported
2. **Colors not updating**: Check CSS specificity and variable scope
3. **Dark mode not working**: Verify the `dark` class is applied to the root element

### Browser Support

Glass effects require modern browser support:
- Chrome 76+
- Firefox 103+
- Safari 14+
- Edge 79+

For older browsers, components will gracefully fallback to solid backgrounds.
