## Technical Features

- **GSAP-Powered**: High-performance animations using the industry-standard GSAP library
- **Liquid Glass Effects**: Morphing shapes and blur effects that simulate liquid behavior
- **Magnetic Hover**: Elements that attract and follow the mouse cursor
- **Scroll Animations**: Trigger animations based on scroll position with intersection observer
- **Accessibility**: Full support for `prefers-reduced-motion` and keyboard navigation
- **Performance Optimized**: GPU-accelerated animations with `transform3d` and `will-change`
