# useContentAwareGlass

A hook that automatically adjusts glass effects based on content and context for optimal visual hierarchy.

## Usage

```tsx
import { useContentAwareGlass } from 'liquidify'

function ContentAwareComponent() {
  const { glassStyle, opacity, blurLevel } = useContentAwareGlass({
    content: 'text',
    importance: 'high',
    context: 'overlay'
  })

  return (
    <div style={glassStyle} className="glass-container">
      <h2>Dynamic Glass Effect</h2>
      <p>This glass adapts to its content automatically.</p>
    </div>
  )
}
```

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `content` | `'text' \| 'image' \| 'mixed'` | `'mixed'` | Type of content behind glass |
| `importance` | `'low' \| 'medium' \| 'high'` | `'medium'` | Content importance level |
| `context` | `'background' \| 'overlay' \| 'modal'` | `'background'` | Usage context |
| `autoAdjust` | `boolean` | `true` | Enable automatic adjustments |

## Return Values

| Property | Type | Description |
|----------|------|-------------|
| `glassStyle` | `CSSProperties` | Dynamic CSS properties for glass effect |
| `opacity` | `number` | Calculated opacity value |
| `blurLevel` | `number` | Calculated blur intensity |
| `contrast` | `number` | Content contrast adjustment |

## Examples

### Text-Heavy Content

```tsx
function TextCard() {
  const { glassStyle } = useContentAwareGlass({
    content: 'text',
    importance: 'high'
  })

  return (
    <div style={glassStyle}>
      <h3>Important Text</h3>
      <p>Glass automatically reduces to ensure readability.</p>
    </div>
  )
}
```

### Image Background

```tsx
function ImageOverlay() {
  const { glassStyle } = useContentAwareGlass({
    content: 'image',
    context: 'overlay',
    importance: 'medium'
  })

  return (
    <div style={glassStyle}>
      <span>Overlay text with adaptive glass</span>
    </div>
  )
}
```
