# Component Showcase

A utility component for displaying interactive examples of LiquidUI components in the documentation. The ComponentShowcase provides a consistent container with glassmorphism styling for component demonstrations.

## Basic Usage

<ComponentShowcase>
  <button className="px-4 py-2 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors">
    Example Button
  </button>
</ComponentShowcase>

## Features

- **Glassmorphism Container**: Provides a beautiful glass-effect background for component examples
- **Responsive Design**: Automatically adapts to different screen sizes
- **Consistent Styling**: Ensures all component examples have uniform presentation
- **Interactive Demos**: Allows users to interact with components directly in the documentation

## Implementation

The ComponentShowcase component is implemented in `/components/ComponentShowcase.tsx` and provides:

```tsx
interface ComponentShowcaseProps {
  children: React.ReactNode;
  className?: string;
}
```

### Default Styling

The showcase container includes:
- Glassmorphism background with `backdrop-blur-md bg-white/10`
- Subtle border with `border border-white/20`
- Rounded corners with `rounded-2xl`
- Responsive padding and centering
- Interactive hover effects

## Best Practices

### Do's
- Use ComponentShowcase for all interactive component examples
- Keep examples focused on demonstrating specific component features
- Include multiple variants when showing component flexibility
- Use semantic HTML structure within examples

### Don'ts
- Don't use ComponentShowcase for code snippets (use regular code blocks)
- Avoid overly complex examples that distract from the component being demonstrated
- Don't nest ComponentShowcase components

## Styling Guidelines

The ComponentShowcase is designed to:
- Provide consistent visual presentation across all documentation pages
- Maintain the glassmorphism design theme of LiquidUI
- Ensure sufficient contrast for component visibility
- Support both light and dark theme contexts

## Accessibility

The ComponentShowcase component includes:
- Semantic container structure
- Appropriate focus management for interactive elements
- Screen reader compatible markup
- Keyboard navigation support for contained components
