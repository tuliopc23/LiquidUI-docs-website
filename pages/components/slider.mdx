# Slider

A slider is a form element that allows the user to select a value from a range.

<ComponentShowcase>
  <GlassSlider defaultValue={[50]} max={100} step={1} />
</ComponentShowcase>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `defaultValue` | `number[]` | `[]` | The default value of the slider. |
| `value` | `number[]` | `undefined` | The value of the slider. |
| `onValueChange` | `(value: number[]) => void` | `undefined` | A callback that is called when the value of the slider changes. |
| `max` | `number` | `100` | The maximum value of the slider. |
| `step` | `number` | `1` | The step of the slider. |
