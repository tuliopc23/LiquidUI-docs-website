## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `React.ReactNode` | - | Main hero title content |
| `subtitle` | `React.ReactNode` | - | Hero subtitle or description |
| `actions` | `React.ReactNode` | - | Call-to-action buttons or elements |
| `background` | `"none" \| "gradient" \| "particles" \| "waves"` | `"none"` | Background animation type |
| `variant` | `"default" \| "minimal" \| "elevated" \| "fullscreen"` | `"default"` | Visual style variant |
| `size` | `"sm" \| "md" \| "lg" \| "xl"` | `"lg"` | Hero section size |
| `alignment` | `"left" \| "center" \| "right"` | `"center"` | Content alignment |
| `className` | `string` | - | Additional CSS classes |

### Physics Properties

The GlassHero includes advanced interactive features:

- **Parallax Effects**: Multi-layer background movement on scroll
- **Particle Animation**: Floating glass particles with realistic physics
- **Adaptive Blur**: Dynamic blur adjustment based on content
- **Magnetic Interactions**: Buttons that respond to cursor proximity
- **Liquid Morphing**: Smooth shape transitions and hover effects

### Background Variants

- **None**: Clean hero without background effects
- **Gradient**: Animated gradient background with color transitions
- **Particles**: Floating glass particles with physics simulation
- **Waves**: Liquid wave animation with glassmorphism effects

### Accessibility

- Semantic HTML structure with proper heading hierarchy
- Screen reader compatible with ARIA labels
- Keyboard navigation support for interactive elements
- High contrast mode support
- Motion preferences respect (reduced motion)

## Usage Guidelines

### Best Practices

- Use clear, compelling headlines that communicate value
- Keep call-to-action buttons prominent and actionable
- Ensure sufficient contrast for text readability
- Consider mobile responsiveness for all content
- Use background effects sparingly to avoid distraction

### When to Use

- **Landing Pages**: Main website entry point and first impression
- **Product Launch**: Introducing new features or products
- **Campaign Pages**: Marketing and promotional content
- **App Onboarding**: Welcome screens and getting started flows

### Animated Hero with Stats

```tsx
import { GlassHero, GlassButton } from 'liquidify';
import { useState, useEffect } from 'react';

function AnimatedHero() {
  const [stats, setStats] = useState({ users: 0, components: 0, downloads: 0 });

  useEffect(() => {
    // Animate numbers on mount
    const timer = setInterval(() => {
      setStats(prev => ({
        users: Math.min(prev.users + 127, 10000),
        components: Math.min(prev.components + 1, 30),
        downloads: Math.min(prev.downloads + 289, 0)
      }));
    }, 50);

    return () => clearInterval(timer);
  }, []);

  return (
    <GlassHero
      title="Trusted by Developers Worldwide"
              subtitle="Join thousands of developers building beautiful UIs with LiqUIdify"
      background="waves"
      actions={
        <div className="space-y-8">
          <a href="/community">
            <GlassButton variant="primary" size="lg">
              Join the Community
            </GlassButton>
          </a>
          
          <div className="grid grid-cols-3 gap-8 text-center">
            <div>
              <div className="text-3xl font-bold text-blue-600 hero-text">
                {stats.users.toLocaleString()}+
              </div>
              <div className="text-sm text-muted-foreground body-text">Active Users</div>
            </div>
            <div>
              <div className="text-3xl font-bold text-green-600 hero-text">
                {stats.components}+
              </div>
              <div className="text-sm text-muted-foreground body-text">Components</div>
            </div>
            <div>
              <div className="text-3xl font-bold text-purple-600 hero-text">
                Soon
              </div>
              <div className="text-sm text-muted-foreground body-text">Launch Day</div>
            </div>
          </div>
        </div>
      }
    />
  );
}
```
