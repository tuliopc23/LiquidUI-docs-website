## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `src` | `string` | `undefined` | The source of the image. |
| `alt` | `string` | `undefined` | The alt text for the image. |
| `fallback` | `React.ReactNode` | `undefined` | The fallback content to render if the image fails to load. |
