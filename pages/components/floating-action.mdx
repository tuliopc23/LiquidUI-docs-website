## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `React.ReactNode` | - | Main floating action button icon |
| `actions` | `FloatingAction[]` | `[]` | Array of secondary action items |
| `position` | `"bottom-right" \| "bottom-left" \| "top-right" \| "top-left" \| "center"` | `"bottom-right"` | Screen position |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Button size |
| `variant` | `"default" \| "minimal" \| "glow"` | `"default"` | Visual style variant |
| `expandDirection` | `"up" \| "down" \| "left" \| "right" \| "radial"` | `"up"` | Expansion direction |
| `tooltip` | `string` | - | Tooltip text for the main button |
| `enableMagnetic` | `boolean` | `true` | Enable magnetic hover effects |
| `className` | `string` | - | Additional CSS classes |
| `onClick` | `() => void` | - | Click handler for main button |

### FloatingAction Interface

```tsx
interface FloatingAction {
  icon: React.ReactNode;
  label: string;
  onClick: () => void;
  color?: "primary" | "secondary" | "success" | "warning" | "error";
  disabled?: boolean;
}
```

### Physics Properties

The GlassFloatingAction includes advanced interactive features:

- **Magnetic Attraction**: Main button attracts cursor within proximity
- **Liquid Morphing**: Smooth shape transitions during interactions
- **Physics Expansion**: Secondary actions expand with spring physics
- **Adaptive Positioning**: Smart collision detection and repositioning
- **Touch Optimization**: Enhanced touch targets for mobile devices

### Accessibility

- Semantic button structure with proper ARIA labels
- Keyboard navigation support (Tab, Enter, Escape)
- Screen reader compatible with action descriptions
- Focus management for expanded states
- High contrast mode support

## Usage Guidelines

### Best Practices

- Use for primary actions that are frequently accessed
- Limit secondary actions to 3-6 items for usability
- Ensure the main icon clearly represents the primary action
- Position thoughtfully to avoid blocking important content
- Provide meaningful tooltips for better UX

### When to Use

- **Primary Actions**: Quick access to main app functionality
- **Content Creation**: Add, upload, or create new items
- **Social Actions**: Share, like, comment, or bookmark
- **Navigation**: Quick access to important app sections

### Content Creation FAB

```tsx
import { GlassFloatingAction } from 'liquidify';
import { Plus, FileText, Image, Video, Music } from 'lucide-react';

function ContentCreationFAB() {
  const contentActions = [
    {
      icon: <FileText className="w-5 h-5" />,
      label: "New Document",
      onClick: () => createDocument(),
      color: "primary"
    },
    {
      icon: <Image className="w-5 h-5" />,
      label: "Upload Image",
      onClick: () => uploadImage(),
      color: "success"
    },
    {
      icon: <Video className="w-5 h-5" />,
      label: "Record Video",
      onClick: () => recordVideo(),
      color: "warning"
    },
    {
      icon: <Music className="w-5 h-5" />,
      label: "Add Audio",
      onClick: () => addAudio(),
      color: "secondary"
    }
  ];

  return (
    <GlassFloatingAction
      icon={<Plus className="w-6 h-6" />}
      actions={contentActions}
      position="bottom-right"
      expandDirection="up"
      tooltip="Create Content"
      variant="glow"
    />
  );
}
```

### Quick Settings FAB

```tsx
import { GlassFloatingAction } from 'liquidify';
import { Settings, Moon, Sun, Volume2, VolumeX, Wifi } from 'lucide-react';
import { useTheme } from 'liquidify';

function QuickSettingsFAB() {
  const { theme, toggleTheme } = useTheme();
  const [volume, setVolume] = useState(true);
  const [wifi, setWifi] = useState(true);

  const settingsActions = [
    {
      icon: theme === 'dark' ? <Sun className="w-5 h-5" /> : <Moon className="w-5 h-5" />,
      label: `Switch to ${theme === 'dark' ? 'Light' : 'Dark'} Mode`,
      onClick: () => toggleTheme(),
      color: "primary"
    },
    {
      icon: volume ? <Volume2 className="w-5 h-5" /> : <VolumeX className="w-5 h-5" />,
      label: volume ? 'Mute' : 'Unmute',
      onClick: () => setVolume(!volume),
      color: volume ? "success" : "error"
    },
    {
      icon: <Wifi className="w-5 h-5" />,
      label: wifi ? 'Disconnect WiFi' : 'Connect WiFi',
      onClick: () => setWifi(!wifi),
      color: wifi ? "success" : "warning"
    }
  ];

  return (
    <GlassFloatingAction
      icon={<Settings className="w-6 h-6" />}
      actions={settingsActions}
      position="top-right"
      expandDirection="left"
      tooltip="Quick Settings"
      variant="minimal"
    />
  );
}
```

### Mobile-Optimized FAB

```tsx
import { GlassFloatingAction } from 'liquidify';
import { useMobile } from 'liquidify';
import { Menu, Home, Search, User, Bell } from 'lucide-react';

  const isMobile = useMobile();

  if (!isMobile) return null;

    {

      onClick: () => navigate('/'),
      color: "primary"
    },
    {
      icon: <Search className="w-5 h-5" />,
      label: "Search",
      onClick: () => navigate('/search'),
      color: "secondary"
    },
    {
      icon: <Bell className="w-5 h-5" />,
      label: "Notifications",
      onClick: () => navigate('/notifications'),
      color: "warning"
    },
    {
      icon: <User className="w-5 h-5" />,
      label: "Profile",
      onClick: () => navigate('/profile'),
      color: "success"
    }
  ];

  return (
    <GlassFloatingAction
      icon={<Menu className="w-6 h-6" />}
      actions={navigationActions}
      position="bottom-right"

      size="lg"
      tooltip="Quick Navigation"
      enableMagnetic={false} // Disabled for better mobile experience
    />
  );
}
```
