
## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| mainIcon | `React.ReactNode` | — | Main floating action button icon |
| actions | `FloatingAction[]` | `[]` | Array of secondary action items |
| position | `'bottom-right' \| 'bottom-left' \| 'top-right' \| 'top-left' \| 'center'` | `'bottom-right'` | Screen position of the floating action button |
| size | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the floating action button |
| variant | `'default' \| 'minimal' \| 'glow'` | `'default'` | Visual style variant |
| enableMagnetic | `boolean` | `true` | Enable magnetic hover effects |
| expandDirection | `'up' \| 'down' \| 'left' \| 'right' \| 'radial'` | `'up'` | Direction to expand secondary actions |
| tooltip | `string` | — | Optional tooltip text for the main button |
