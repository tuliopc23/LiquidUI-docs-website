# Progress

A progress bar is a visual representation of the progress of an operation.

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `number` | `undefined` | The value of the progress bar. |
| `max` | `number` | `100` | The maximum value of the progress bar. |
