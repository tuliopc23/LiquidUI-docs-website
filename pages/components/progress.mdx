## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Progress value (0-100) |
| `variant` | `"default" \| "success" \| "warning" \| "error" \| "circular" \| "steps"` | `"default"` | Visual variant |
| `size` | `"sm" \| "default" \| "lg"` \| `number` | `"default"` | Size of progress indicator |
| `showLabel` | `boolean` | `false` | Show percentage label |
| `indeterminate` | `boolean` | `false` | Show indeterminate/loading state |
| `color` | `string` | - | Custom color override |
| `className` | `string` | - | Additional CSS classes |
| `animate` | `boolean` | `true` | Enable progress animations |

### Step Progress Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `currentStep` | `number` | `0` | Current active step (0-based) |
| `totalSteps` | `number` | - | Total number of steps |
| `steps` | `Step[]` | - | Array of step configurations |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Step layout direction |

### Accessibility

- **Screen Readers**: Proper ARIA attributes and progress announcements
- **Keyboard Navigation**: Focusable when interactive
- **High Contrast**: Adapts to system high contrast modes
- **Motion**: Respects prefers-reduced-motion settings

## Usage Guidelines

### Best Practices

- Use progress indicators for operations longer than 2 seconds
- Provide clear labels describing what's in progress
- Choose appropriate variants for context
- Keep progress animations smooth and purposeful
- Consider indeterminate progress for unknown durations

### When to Use

- **File uploads/downloads**
- **Form submission progress**
- **Multi-step processes**
- **Loading states with known duration**
- **Onboarding flows**

## Examples

### Dynamic Progress

```tsx
import { GlassProgress } from "glass-ui-tulio";
import { useState, useEffect } from "react";

function DynamicProgress() {
  const [progress, setProgress] = useState(0);
  
  useEffect(() => {
    const timer = setInterval(() => {
      setProgress(prev => {
        if (prev >= 100) return 0; // Reset for demo
        return prev + 10;
      });
    }, 500);
    
    return () => clearInterval(timer);
  }, []);
  
  return (
    <GlassProgress 
      value={progress} 
      variant={progress === 100 ? "success" : "default"}
      showLabel 
    />
  );
}
```

### Upload Progress with Cancel

```tsx
import { GlassProgress, GlassButton } from "glass-ui-tulio";
import { useState } from "react";

function UploadProgress() {
  const [uploading, setUploading] = useState(false);
  const [progress, setProgress] = useState(0);
  
  const startUpload = () => {
    setUploading(true);
    // Simulate upload progress
    const interval = setInterval(() => {
      setProgress(prev => {
        if (prev >= 100) {
          setUploading(false);
          clearInterval(interval);
          return 100;
        }
        return prev + 10;
      });
    }, 200);
  };
  
  const cancelUpload = () => {
    setUploading(false);
    setProgress(0);
  };
  
  return (
    <div className="space-y-4">
      <GlassProgress value={progress} showLabel />
      
      <div className="flex gap-2">
        <GlassButton 
          onClick={startUpload} 
          disabled={uploading}
        >
          {uploading ? 'Uploading...' : 'Start Upload'}
        </GlassButton>
        
        {uploading && (
          <GlassButton 
            variant="secondary" 
            onClick={cancelUpload}
          >
            Cancel
          </GlassButton>
        )}
      </div>
    </div>
  );
}
```
