## Props

| Name | Type | Default | Description |
|------|------|---------|-------------|
| title | `string` | — | Optional header title for the showcase |
| subtitle | `string` | — | Optional subtitle displayed above the title |
| description | `string` | — | Optional description text below the title |
| features | `FeatureItem[]` | — | Array of feature items to display |
| layout | `'grid' \| 'masonry' \| 'carousel' \| 'stacked'` | `'grid'` | Layout style for displaying features |
| columns | `1 \| 2 \| 3 \| 4` | `3` | Number of columns in grid layout |
| variant | `'default' \| 'minimal' \| 'cards' \| 'floating'` | `'default'` | Visual style variant |
| enableMagnetic | `boolean` | `false` | Enable magnetic hover effects |
| enableParallax | `boolean` | `true` | Enable parallax background animations |
