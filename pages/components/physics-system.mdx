# Physics System

The LiqUIdify physics system provides advanced particle simulation, fluid dynamics, and interactive physics effects for creating immersive liquid glass experiences.

## Core Components

### FluidParticle

Individual fluid particles with physics properties.

```tsx
import { FluidParticle } from 'liquidify'

function ParticleDemo() {
  return (
    <FluidParticle
      position={{ x: 100, y: 100 }}
      velocity={{ x: 0, y: 0 }}
      mass={1}
      radius={5}
    />
  )
}
```

### FluidSimulation

Complete fluid simulation container with multiple particles.

```tsx
import { FluidSimulation } from 'liquidify'

function FluidDemo() {
  return (
    <FluidSimulation
      particleCount={50}
      containerWidth={800}
      containerHeight={600}
      viscosity={0.8}
      gravity={{ x: 0, y: 9.8 }}
    >
      Your content here
    </FluidSimulation>
  )
}
```

### PhysicsWorld

Physics world container for managing all physics objects.

```tsx
import { PhysicsWorld } from 'liquidify'

function PhysicsDemo() {
  return (
    <PhysicsWorld
      gravity={{ x: 0, y: 9.8 }}
      bounds={{ width: 1000, height: 800 }}
    >
      <FluidSimulation />
      <ParticleEmitter />
    </PhysicsWorld>
  )
}
```

## Physics Utilities

### Spring2D

2D spring physics for smooth animations.

```tsx
import { Spring2D, SPRING_PRESETS } from 'liquidify'

const spring = new Spring2D({
  tension: SPRING_PRESETS.gentle.tension,
  friction: SPRING_PRESETS.gentle.friction
})
```

### Vector2D

2D vector mathematics for physics calculations.

```tsx
import { Vector2D } from 'liquidify'

const velocity = new Vector2D(10, -5)
const acceleration = new Vector2D(0, 9.8)

// Vector operations
const newVelocity = velocity.add(acceleration)
const magnitude = velocity.magnitude()
```

## Performance Considerations

The physics system is optimized for performance but consider:

- Limit particle count for mobile devices
- Use `useGlassEffectPerformance` hook to monitor performance
- Enable GPU acceleration for smooth animations
- Consider reducing physics complexity on lower-end devices
