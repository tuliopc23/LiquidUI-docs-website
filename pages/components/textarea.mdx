## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `rows` | `number` | `4` | Number of visible text lines |
| `cols` | `number` | - | Number of visible character columns |
| `size` | `"sm" \| "default" \| "lg"` | `"default"` | The size of the textarea |
| `placeholder` | `string` | - | Placeholder text |
| `value` | `string` | - | The controlled value |
| `defaultValue` | `string` | - | The default value for uncontrolled usage |
| `disabled` | `boolean` | `false` | Whether the textarea is disabled |
| `required` | `boolean` | `false` | Whether the textarea is required |
| `maxLength` | `number` | - | Maximum number of characters allowed |
| `autoResize` | `boolean` | `false` | Enable automatic height adjustment |
| `className` | `string` | - | Additional CSS classes |
| `onChange` | `(event: ChangeEvent<HTMLTextAreaElement>) => void` | - | Change event handler |
| `onFocus` | `(event: FocusEvent<HTMLTextAreaElement>) => void` | - | Focus event handler |
| `onBlur` | `(event: FocusEvent<HTMLTextAreaElement>) => void` | - | Blur event handler |

### Advanced Features

- **Auto-resize**: Automatically adjusts height based on content
- **Character counting**: Built-in character limit tracking
- **Glassmorphism effects**: Backdrop blur and transparency
- **Content-aware styling**: Adapts to background content

### Accessibility

- Full keyboard navigation support
- Screen reader compatible with proper labeling
- Focus management and visual indicators
- High contrast mode support
- ARIA attributes for validation states

## Usage Guidelines

### Best Practices

- Provide clear labels that describe the expected content
- Use appropriate `rows` attribute for expected content length
- Include character limits when necessary
- Provide helpful validation feedback
- Consider auto-resize for dynamic content

### When to Use

- **Comments**: User feedback and comments
- **Descriptions**: Product or project descriptions  
- **Messages**: Contact forms and messaging
- **Reviews**: User reviews and testimonials
- **Notes**: Multi-line notes and observations

## Examples

### Controlled Textarea

```tsx
import { GlassTextarea } from "../../components/GlassComponents";
import { useState } from "react";

function ControlledTextarea() {
  const [value, setValue] = useState("");

  return (
    <GlassTextarea
      value={value}
      onChange={(e) => setValue(e.target.value)}
      placeholder="Start typing..."
      rows={4}
    />
  );
}
```

### With Character Count

```tsx
import { GlassTextarea } from "../../components/GlassComponents";
import { useState } from "react";

function TextareaWithCount() {
  const [value, setValue] = useState("");
  const maxLength = 500;

  return (
    <div>
      <div className="flex justify-between items-center mb-2">
        <label className="text-sm font-medium">Description</label>
        <span className="text-xs text-muted-foreground">
          {value.length}/{maxLength}
        </span>
      </div>
      <GlassTextarea
        value={value}
        onChange={(e) => setValue(e.target.value)}
        maxLength={maxLength}
        rows={4}
        placeholder="Describe your project..."
      />
    </div>
  );
}
```
