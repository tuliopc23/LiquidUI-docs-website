# Glass Gestures

Touch and pointer gesture support for LiquidUI glass components. Create intuitive interactions that feel natural on all devices.

## Overview

Glass Gestures provide comprehensive touch and pointer interaction patterns optimized for glassmorphism interfaces, including swipe, pinch, drag, and complex multi-touch gestures.

## Gesture Types

### Swipe Gestures

```typescript
import { useSwipeGesture } from 'liquidify';

function SwipeableGlassCard() {
  const { onTouchStart, onTouchMove, onTouchEnd } = useSwipeGesture({
    onSwipeLeft: () => console.log('Swiped left'),
    onSwipeRight: () => console.log('Swiped right'),
    threshold: 50
  });
  
  return (
    <div
      className="backdrop-blur-md bg-white/20 p-6 rounded-xl"
      onTouchStart={onTouchStart}
      onTouchMove={onTouchMove}
      onTouchEnd={onTouchEnd}
    >
      Swipe me left or right
    </div>
  );
}
```

### Pinch to Zoom

```tsx
import { usePinchGesture } from 'liquidify';

function PinchableGlass() {
  const { scale, onTouchStart, onTouchMove } = usePinchGesture({
    minScale: 0.5,
    maxScale: 3,
    onScaleChange: (scale) => console.log('Scale:', scale)
  });
  
  return (
    <div
      className="backdrop-blur-md bg-white/20 rounded-xl overflow-hidden"
      style={{ transform: `scale(${scale})` }}
      onTouchStart={onTouchStart}
      onTouchMove={onTouchMove}
    >
      Pinch to zoom
    </div>
  );
}
```

## Mobile Optimizations

Glass gestures are optimized for mobile performance:

```css
.glass-gesture {
  touch-action: manipulation;
  -webkit-tap-highlight-color: transparent;
  user-select: none;
}
```
