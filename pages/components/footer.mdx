# Footer

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `links` | `FooterLink[]` | - | Array of footer navigation links |
| `sections` | `FooterSection[]` | - | Array of footer sections for multi-column layout |
| `copyright` | `string` | - | Copyright text to display |
| `variant` | `"default" \| "minimal" \| "social" \| "columns"` | `"default"` | Visual style variant |
| `className` | `string` | - | Additional CSS classes |

### FooterLink Interface

```tsx
interface FooterLink {
  label: string;
  href: string;
  icon?: string;
  external?: boolean;
}
```

### FooterSection Interface

```tsx
interface FooterSection {
  title: string;
  links: FooterLink[];
}
```

### Physics Properties

The GlassFooter includes advanced glassmorphism features:

- **Adaptive Background**: Automatically adjusts opacity based on page content
- **Liquid Hover Effects**: Smooth transitions on link interactions
- **Content-Aware Styling**: Optimizes glass effects for better readability
- **Magnetic Link Attraction**: Subtle hover effects that feel natural

### Accessibility

- Semantic HTML structure with proper landmark roles
- Screen reader compatible with ARIA labels
- Keyboard navigation support for all links
- High contrast mode support
- Focus management and visual indicators

## Usage Guidelines

### Best Practices

- Keep link organization logical and intuitive
- Limit the number of footer links to avoid clutter
- Ensure links are easily clickable on mobile devices
- Use consistent spacing and typography
- Consider legal requirements for footer content

### When to Use

- **Site Navigation**: Secondary navigation and utility links
- **Legal Pages**: Privacy policy, terms of service, accessibility
- **Social Media**: Links to social platforms and community
- **Contact Information**: Support, contact, and help resources

## Examples

### Complete Application Footer

```tsx
import { GlassFooter } from 'liquidify';

function AppFooter() {
  const footerSections = [
    {
      title: "Product",
      links: [
        { label: "Features", href: "/features" },
        { label: "Pricing", href: "/pricing" },
        { label: "Documentation", href: "/components" },
        { label: "API Reference", href: "/api" }
      ]
    },
    {
      title: "Community",
      links: [
        { label: "GitHub Issues", href: "https://github.com/tuliocunha23/liquidui/issues", external: true },
        { label: "GitHub", href: "https://github.com/tuliocunha23/liquidui", external: true },
        { label: "Twitter", href: "https://twitter.com/liquidui", external: true },
        { label: "Blog", href: "/blog" }
      ]
    },
    {
      title: "Support",
      links: [
        { label: "Help Center", href: "/help" },
        { label: "Contact", href: "/contact" },
        { label: "Status", href: "/status" },
        { label: "Feedback", href: "/feedback" }
      ]
    },
    {
      title: "Legal",
      links: [
        { label: "Privacy Policy", href: "/privacy" },
        { label: "Terms of Service", href: "/terms" },
        { label: "Cookie Policy", href: "/cookies" },
        { label: "Accessibility", href: "/accessibility" }
      ]
    }
  ];

  return (
    <GlassFooter
      sections={footerSections}
                copyright="© 2024 LiqUIdify. Built with liquid glass components."
      variant="columns"
    />
  );
}
```
