# Button

A button is a clickable element that is used to trigger an action.

<ComponentShowcase>
  <GlassButton>Default</GlassButton>
  <GlassButton variant="secondary">Secondary</GlassButton>
  <GlassButton variant="destructive">Destructive</GlassButton>
  <GlassButton variant="ghost">Ghost</GlassButton>
  <GlassButton variant="link">Link</GlassButton>
</ComponentShowcase>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"default" \| "secondary" \| "destructive" \| "ghost" \| "link"` | `"default"` | The variant of the button. |
| `size` | `"default" \| "sm" \| "lg" \| "icon"` | `"default"` | The size of the button. |
| `asChild` | `boolean` | `false` | Whether to render the button as a child element. |
