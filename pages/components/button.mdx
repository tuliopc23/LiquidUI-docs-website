## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "destructive" \| "ghost" \| "link"` | `"default"` | The visual style variant of the button |
| `size` | `"default" \| "sm" \| "lg" \| "icon"` | `"default"` | The size of the button |
| `disabled` | `boolean` | `false` | Whether the button is disabled |
| `loading` | `boolean` | `false` | Whether the button shows a loading state |
| `asChild` | `boolean` | `false` | Render as a child element (useful for links) |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Button content |
| `onClick` | `(event: MouseEvent) => void` | - | Click event handler |

### Physics Properties

The GlassButton includes advanced physics-based interactions:

- **Magnetic Hover**: Buttons attract the cursor within a proximity radius
- **Spring Animation**: Smooth enter/exit animations with spring physics
- **Content Awareness**: Adapts glass opacity based on background content
- **Touch Response**: Optimized haptic feedback for mobile devices

### Accessibility

- Full keyboard navigation support
- Screen reader compatible with proper ARIA labels
- Focus management and visual indicators
- High contrast mode support

## Usage Guidelines

### Best Practices

- Use primary buttons for main actions
- Limit to one primary button per section
- Provide clear, action-oriented labels
- Consider loading states for async actions

### When to Use

- **Default**: Primary actions, form submissions
- **Secondary**: Secondary actions, cancel operations
- **Destructive**: Delete, remove, or destructive actions
- **Ghost**: Subtle actions, toolbar buttons
- **Link**: Navigation, external links

## Examples

### With Form

```tsx
import { GlassButton } from "glass-ui-tulio";

function LoginForm() {
  return (
    <form className="space-y-4">
      <input type="email" placeholder="Email" />
      <input type="password" placeholder="Password" />
      <div className="flex gap-2">
        <GlassButton type="submit">
          Sign In
        </GlassButton>
        <GlassButton variant="ghost" type="button">
          Cancel
        </GlassButton>
      </div>
    </form>
  );
}
```

### Async Action

```tsx
import { GlassButton } from "glass-ui-tulio";
import { useState } from "react";

function AsyncButton() {
  const [loading, setLoading] = useState(false);

  const handleSubmit = async () => {
    setLoading(true);
    try {
      await submitData();
    } finally {
      setLoading(false);
    }
  };

  return (
    <GlassButton loading={loading} onClick={handleSubmit}>
      {loading ? "Saving..." : "Save Changes"}
    </GlassButton>
  );
}
```
