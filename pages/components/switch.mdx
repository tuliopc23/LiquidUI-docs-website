# Switch

A switch is a form element that allows the user to toggle a value.

<ComponentShowcase>
  <GlassSwitch />
</ComponentShowcase>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `checked` | `boolean` | `false` | Whether the switch is checked. |
| `onCheckedChange` | `(checked: boolean) => void` | `undefined` | A callback that is called when the switch is checked or unchecked. |
