## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `React.ReactNode` | - | Main header title content |
| `subtitle` | `React.ReactNode` | - | Optional subtitle displayed below the title |
| `actions` | `React.ReactNode` | - | Optional action elements (buttons, dropdowns, etc.) |
| `className` | `string` | - | Additional CSS classes |
| `variant` | `"default" \| "minimal" \| "elevated"` | `"default"` | Visual style variant |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Header size |

### Physics Properties

The GlassHeader includes advanced glassmorphism features:

- **Adaptive Blur**: Automatically adjusts blur intensity based on background content
- **Liquid Morphing**: Smooth shape transitions on hover and interaction
- **Content Awareness**: Optimizes glass opacity for better text readability
- **Magnetic Interactions**: Subtle attraction effects for interactive elements

### Accessibility

- Semantic HTML structure with proper heading hierarchy
- Screen reader compatible with ARIA labels
- Keyboard navigation support for action elements
- High contrast mode support

## Usage Guidelines

### Best Practices

- Use clear, descriptive titles that reflect the page content
- Keep action elements to essential functions only
- Ensure sufficient contrast for text readability
- Consider mobile responsiveness for action layouts

### When to Use

- **Page Headers**: Main navigation and page identification
- **Section Headers**: Dividing content into logical sections  
- **Dashboard Headers**: Application status and user controls
- **Modal Headers**: Dialog and overlay identification

## Examples

### Application Header

```tsx
import { GlassHeader, GlassButton } from '@tuliocunha23/liquidui';
import { Search, Bell, User } from 'lucide-react';

function AppHeader() {
  return (
    <GlassHeader
      title="My Application"
      subtitle="Dashboard Overview"
      actions={
        <div className="flex items-center gap-3">
          <GlassButton variant="ghost" size="sm">
            <Search className="w-4 h-4" />
          </GlassButton>
          <GlassButton variant="ghost" size="sm">
            <Bell className="w-4 h-4" />
          </GlassButton>
          <GlassButton variant="secondary" size="sm">
            <User className="w-4 h-4 mr-2" />
            Profile
          </GlassButton>
        </div>
      }
    />
  );
}
```

### Responsive Header

```tsx
import { GlassHeader, GlassButton } from '@tuliocunha23/liquidui';
import { Menu, X } from 'lucide-react';
import { useState } from 'react';

function ResponsiveHeader() {
  const [isMenuOpen, setIsMenuOpen] = useState(false);

  return (
    <GlassHeader
      title="Responsive App"
      variant="elevated"
      actions={
        <div className="flex items-center gap-2">
          <div className="hidden md:flex items-center gap-3">
            <GlassButton variant="ghost">Features</GlassButton>
            <GlassButton variant="ghost">Pricing</GlassButton>
            <GlassButton variant="primary">Get Started</GlassButton>
          </div>
          <GlassButton 
            variant="ghost" 
            size="sm"
            className="md:hidden"
            onClick={() => setIsMenuOpen(!isMenuOpen)}
          >
            {isMenuOpen ? <X className="w-4 h-4" /> : <Menu className="w-4 h-4" />}
          </GlassButton>
        </div>
      }
    />
  );
}
```
