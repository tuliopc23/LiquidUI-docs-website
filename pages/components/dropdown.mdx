# Dropdown

A beautiful glassmorphism dropdown component that displays a list of interactive items with support for icons, separators, and flexible positioning.

## Basic Usage

## TypeScript Interface

```typescript
interface GlassDropdownProps {
  trigger: React.ReactNode;
  items: Array<{
    id: string;
    label: string;
    icon?: React.ReactNode;
    onClick?: () => void;
    disabled?: boolean;
    separator?: boolean;
  }>;
  position?: "bottom-left" | "bottom-right" | "top-left" | "top-right";
  className?: string;
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `trigger` | `React.ReactNode` | - | The element that triggers the dropdown when clicked |
| `items` | `DropdownItem[]` | `[]` | Array of dropdown items to display |
| `position` | `'bottom-left' \| 'bottom-right' \| 'top-left' \| 'top-right'` | `'bottom-left'` | Position of the dropdown relative to trigger |
| `className` | `string` | - | Additional CSS classes for custom styling |

### Item Configuration

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `id` | `string` | ✓ | Unique identifier for the item |
| `label` | `string` | ✓ | Display text for the item |
| `icon` | `React.ReactNode` | - | Optional icon to display before the label |
| `onClick` | `() => void` | - | Function to call when item is clicked |
| `disabled` | `boolean` | - | Whether the item is disabled |
| `separator` | `boolean` | - | Renders as a visual separator line |

## Advanced Usage

### Context Menu Pattern

```tsx
import { GlassDropdown } from 'liquidify';
import { useState } from 'react';

function ContextMenuExample() {
  const [selectedItem, setSelectedItem] = useState<string | null>(null);

  const contextMenuItems = [
    {
      id: "copy",
      label: "Copy",
      onClick: () => {
        navigator.clipboard.writeText("Copied text");
        setSelectedItem("copy");
      }
    },
    {
      id: "paste",
      label: "Paste",
      onClick: async () => {
        const text = await navigator.clipboard.readText();
        console.log("Pasted:", text);
        setSelectedItem("paste");
      }
    },
    { id: "separator", label: "", separator: true },
    {
      id: "delete",
      label: "Delete",
      onClick: () => {
        console.log("Deleting item");
        setSelectedItem("delete");
      }
    }
  ];

  return (
    <div className="p-4 border border-gray-200 rounded-lg">
      <p>Right-click simulation with dropdown:</p>
      <GlassDropdown
        trigger={<div className="mt-2 p-2 bg-gray-100 rounded cursor-pointer">Right-click me</div>}
        items={contextMenuItems}
        position="bottom-right"
      />
      {selectedItem && <p className="mt-2 text-sm text-gray-600">Last action: {selectedItem}</p>}
    </div>
  );
}
```

### Nested Menu Pattern

```tsx
import { GlassDropdown } from 'liquidify';
import { ChevronRight } from 'lucide-react';

function NestedMenuExample() {
  const [showSubmenu, setShowSubmenu] = useState(false);

  return (
    <div className="relative">
      <GlassDropdown
        trigger={<button className="px-4 py-2 bg-blue-500 text-white rounded">File Menu</button>}
        items={[
          { id: "new", label: "New File" },
          { id: "open", label: "Open File" },
          { id: "separator1", label: "", separator: true },
          { 
            id: "export",
            label: "Export As",
            icon: <ChevronRight size={16} />,
            onClick: () => setShowSubmenu(!showSubmenu)
          }
        ]}
      />
      
      {showSubmenu && (
        <div className="absolute left-32 top-0">
          <GlassDropdown
            trigger={<div></div>}
            items={[
              { id: "pdf", label: "Export as PDF" },
              { id: "png", label: "Export as PNG" },
              { id: "svg", label: "Export as SVG" }
            ]}
          />
        </div>
      )}
    </div>
  );
}
```

## Accessibility

The GlassDropdown component includes several accessibility features:

- **Keyboard Navigation**: Use Tab to focus the trigger, Enter/Space to open
- **Click Outside**: Click outside the dropdown to close it
- **Disabled States**: Properly handles disabled items with visual feedback
- **Focus Management**: Maintains focus flow when dropdown opens/closes

### Accessibility Best Practices

```tsx
import { GlassDropdown } from 'liquidify';

function AccessibleDropdown() {
  return (
    <GlassDropdown
      trigger={
        <button 
          className="px-4 py-2 bg-blue-500 text-white rounded"
          aria-label="Open user menu"
          aria-haspopup="true"
        >
          User Menu
        </button>
      }
      items={[
        { 
          id: "profile", 
          label: "Profile", 
          onClick: () => {
            // Announce action to screen readers
            console.log("Profile page opened");
          }
        }
      ]}
    />
  );
}
```

## Styling

The dropdown uses glassmorphism design with these default styles:
- `backdrop-blur-xl` for the glass effect
- `bg-white/20` for semi-transparent background
- `border-white/30` for subtle borders
- `rounded-2xl` for rounded corners
- `shadow-lg` for depth

### Status Dropdown

```tsx
function StatusDropdown() {
  const [status, setStatus] = useState("online");
  
  return (
    <GlassDropdown
      trigger={
        <div className="flex items-center gap-2 cursor-pointer">
          <div className={`w-3 h-3 rounded-full ${status === 'online' ? 'bg-green-500' : 'bg-gray-500'}`}></div>
          <span className="capitalize">{status}</span>
        </div>
      }
      items={[
        { 
          id: "online", 
          label: "Online", 
          onClick: () => setStatus("online")
        },
        { 
          id: "away", 
          label: "Away", 
          onClick: () => setStatus("away")
        },
        { 
          id: "busy", 
          label: "Busy", 
          onClick: () => setStatus("busy")
        },
        { 
          id: "offline", 
          label: "Offline", 
          onClick: () => setStatus("offline")
        }
      ]}
    />
  );
}
```
