# Checkbox

A checkbox is a form element that allows the user to select one or more options.

<ComponentShowcase>
  <GlassCheckbox />
</ComponentShowcase>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `checked` | `boolean` | `false` | Whether the checkbox is checked. |
| `onCheckedChange` | `(checked: boolean) => void` | `undefined` | A callback that is called when the checkbox is checked or unchecked. |
