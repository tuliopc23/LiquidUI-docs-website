# Tooltip

A tooltip is a small pop-up that displays information about an element.

<ComponentShowcase>
  <GlassTooltip content="This is a tooltip">
    <span>Hover me</span>
  </GlassTooltip>
</ComponentShowcase>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `content` | `React.ReactNode` | `undefined` | The content of the tooltip. |
