# Tooltip

A beautiful glassmorphism tooltip component that displays contextual information when hovering over elements. Features smooth animations, multiple positioning options, and elegant glass styling.

## Basic Usage

<ComponentShowcase>
  <div className="flex items-center gap-6 p-8">
    <GlassTooltip content="This is a basic tooltip">
      <button className="px-4 py-2 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors">
        Hover me
      </button>
    </GlassTooltip>
    
    <GlassTooltip content="Tooltip with an icon">
      <Info className="w-5 h-5 text-blue-500 cursor-help" />
    </GlassTooltip>
  </div>
</ComponentShowcase>

## TypeScript Interface

```typescript
interface GlassTooltipProps {
  content: string;
  children: React.ReactNode;
  position?: "top" | "bottom" | "left" | "right";
  className?: string;
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `content` | `string` | - | The text content to display in the tooltip |
| `children` | `React.ReactNode` | - | The element that triggers the tooltip on hover |
| `position` | `"top" \| "bottom" \| "left" \| "right"` | `"top"` | Position of the tooltip relative to the trigger element |
| `className` | `string` | - | Additional CSS classes for custom styling |

### Conditional Tooltips

```tsx
import { GlassTooltip } from 'liquidify';
import { useState } from 'react';

function ConditionalTooltip() {
  const [showTooltip, setShowTooltip] = useState(true);
  
  const TooltipWrapper = ({ children }: { children: React.ReactNode }) => {
    if (showTooltip) {
      return (
        <GlassTooltip content="This tooltip can be toggled">
          {children}
        </GlassTooltip>
      );
    }
    return <>{children}</>;
  };
  
  return (
    <div className="space-y-4">
      <label className="flex items-center gap-2">
        <input
          type="checkbox"
          checked={showTooltip}
          onChange={(e) => setShowTooltip(e.target.checked)}
        />
        Show tooltip
      </label>
      
      <TooltipWrapper>
        <button className="px-4 py-2 bg-blue-500 text-white rounded-lg">
          Hover me
        </button>
      </TooltipWrapper>
    </div>
  );
}
```

## Accessibility

The GlassTooltip component provides basic accessibility features:

- **Hover Activation**: Uses standard mouse hover events

- **Keyboard Navigation**: Works with keyboard focus on interactive elements

## Limitations

The current implementation has some limitations:

1. **Content Type**: Only supports string content, not rich HTML/React nodes
2. **No Delay Configuration**: No built-in delay settings for show/hide
3. **Limited Trigger Events**: Only supports hover, no click or focus triggers
4. **No Animation Control**: Fixed transition timing and style

### Workarounds

```tsx
import { GlassTooltip } from 'liquidify';
import { useState } from 'react';

// Custom delay implementation
function DelayedTooltip({ children, content, delay = 500 }: {
  children: React.ReactNode;
  content: string;
  delay?: number;
}) {
  const [showTooltip, setShowTooltip] = useState(false);
  const [timeoutId, setTimeoutId] = useState<NodeJS.Timeout | null>(null);
  
  const handleMouseEnter = () => {
    const id = setTimeout(() => setShowTooltip(true), delay);
    setTimeoutId(id);
  };
  
  const handleMouseLeave = () => {
    if (timeoutId) {
      clearTimeout(timeoutId);
      setTimeoutId(null);
    }
    setShowTooltip(false);
  };
  
  return (
    <div
      onMouseEnter={handleMouseEnter}
      onMouseLeave={handleMouseLeave}
    >
      {showTooltip ? (
        <GlassTooltip content={content}>
          {children}
        </GlassTooltip>
      ) : (
        children
      )}
    </div>
  );
}
```
