# Popover

A popover is a component that is displayed on top of another component.

<ComponentShowcase>
  <GlassPopover content="This is a popover">
    <span>Hover me</span>
  </GlassPopover>
</ComponentShowcase>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `content` | `React.ReactNode` | `undefined` | The content of the popover. |
