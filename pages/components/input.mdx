# Input

An input is a form element that allows the user to enter text.

<ComponentShowcase>
  <GlassInput />
</ComponentShowcase>

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `type` | `string` | `"text"` | The type of the input. |
| `placeholder` | `string` | `undefined` | The placeholder text for the input. |
