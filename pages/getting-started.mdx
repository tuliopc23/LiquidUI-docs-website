## Installation

### Using npm

```bash
npm install @tuliocunha23/liquidui
```

### Using yarn

```bash
yarn add @tuliocunha23/liquidui
```

### Using pnpm

```bash
pnpm add @tuliocunha23/liquidui
```

## Setup

### 1. Import Styles

Add the LiquidiUI CSS to your project. In your main CSS file or `_app.tsx`:

```tsx
import '@tuliocunha23/liquidui/css'
```

### 2. Configure Tailwind CSS

If you're using Tailwind CSS (recommended), add LiquidiUI to your `tailwind.config.js`:

```js
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './pages/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',
    './app/**/*.{js,ts,jsx,tsx,mdx}',
    './node_modules/@tuliocunha23/liquidui/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

### 3. Setup Theme Provider (Optional)

For better theme support, wrap your app with the theme provider:

```tsx
import { ThemeProvider } from '@tuliocunha23/liquidui'

function MyApp({ Component, pageProps }) {
  return (
    <ThemeProvider>
      <Component {...pageProps} />
    </ThemeProvider>
  )
}
```

## Key Features

### Physics-Based Interactions

LiquidiUI components respond to user interaction with natural, physics-based animations:

- **Magnetic Hover**: Components attract the cursor within a proximity radius
- **Spring Animations**: Smooth, natural transitions using spring physics
- **Gesture Recognition**: Advanced touch and mouse gesture support

### Content-Aware Adaptation

Components automatically adapt to their environment:

- **Background Detection**: Adjusts opacity based on background content
- **Color Contrast**: Maintains readability across different backgrounds
- **Dynamic Sizing**: Responds to content changes with smooth transitions

### Accessibility First

Built with accessibility as a core principle:

- **Screen Reader Support**: Full ARIA compliance
- **Keyboard Navigation**: Complete keyboard interaction support
- **Focus Management**: Intelligent focus handling and indicators
- **High Contrast**: Supports system high contrast modes

## TypeScript Support

LiquidiUI is built with TypeScript and provides full type safety:

```tsx
import { GlassButtonProps } from '@tuliocunha23/liquidui'

interface MyButtonProps extends GlassButtonProps {
  customProp?: string
}

function MyButton({ customProp, ...props }: MyButtonProps) {
  return <GlassButton {...props} />
}
```

## Performance

LiquidiUI is optimized for performance:

- **Tree Shaking**: Only import components you use
- **Lazy Loading**: Components load on demand
- **Minimal Bundle Size**: Efficient packaging and compression
- **Hardware Acceleration**: Uses GPU acceleration for smooth animations

## Browser Support

LiquidiUI supports all modern browsers:
