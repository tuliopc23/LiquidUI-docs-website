```typescript

  title: "Success!",
  description: "Your action was completed",
  variant: "success", // 'success' | 'error' | 'warning' | 'info'
  duration: 5000,
  position: "top-right" // 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right'
});
```

### useModal

### Date Formatting

```typescript
import { formatDate } from '@tuliocunha23/liquidui';

formatDate(new Date()); // "Jan 1, 2024"
formatDate(new Date(), 'full'); // "Monday, January 1, 2024"
formatDate(new Date(), 'time'); // "12:00 PM"
formatDate(new Date(), 'datetime'); // "Jan 1, 2024 at 12:00 PM"
```

### Debounce & Throttle

```typescript
import { debounce, throttle } from '@tuliocunha23/liquidui';

// Debounce - delays execution until after wait time
const debouncedSearch = debounce((query: string) => {
  console.log('Searching:', query);
}, 500);

// Throttle - limits execution to once per wait time
const throttledScroll = throttle(() => {
  console.log('Scrolling');
}, 100);
```

## üéØ Best Practices

### Performance

1. **Tree Shaking**: Import only what you need
   ```typescript
   // ‚úÖ Good
   import { GlassButton, GlassCard } from '@tuliocunha23/liquidui';
   
   // ‚ùå Avoid
       import * as LiqUIdify from '@tuliocunha23/liquidui';
   ```

2. **Lazy Loading**: Use dynamic imports for heavy components
   ```typescript
   const GlassChart = dynamic(() => 
     import('@tuliocunha23/liquidui').then(mod => mod.GlassChart)
   );
   ```

3. **Memoization**: Wrap expensive components
   ```typescript
   const MemoizedCard = React.memo(GlassCard);
   ```

### Accessibility

All components follow WCAG 2.1 guidelines:

- Proper ARIA attributes
- Keyboard navigation support
- Focus management
- Screen reader announcements
- Color contrast compliance

Components accept standard CSS classes and style props:

```tsx
  data-testid="submit-button"
/>
```
